LWAVEH5 =  Default: LWAVEH5 = LH5 Description: Determines whether the wavefunctions are written to vaspwave.h5 file at the end of a run. If LWAVEH5 = True , the Kohn-Sham orbitals, i.e., wavefunctions, are written to the vaspwave.h5 file at the end of the calculation. This can be used to restart VASP.
