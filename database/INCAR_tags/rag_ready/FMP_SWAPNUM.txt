FMP_SWAPNUM = integer Default: FMP_SWAPNUM = 1 Description: Number of pairs that are exchanged in a single swapping event in the Müller-Plathe method . FMP_SWAPNUM defines the number of pairs of particles exchanged in a single swapping event of the reverse nonequilibrium molecular dynamics run using the Müller-Plathe method . Only the particles of the same type are selected. Related tags and articles Müller-Plathe method , FMP_DIRECTION , FMP_ACTIVE , FMP_SNUMBER , FMP_PERIOD
