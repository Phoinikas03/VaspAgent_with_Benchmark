IRC_VNORM0 = real Default: IRC_VNORM0 = 0.002 Description: The constant velocity vector (in Angstrom/fs) in an IRC calculation . In the damped-velocity-Verlet algorithm for IRC calculations , the damping is realized via rescaling the velocity vector to a constant value (in Angstrom/fs) defined via the parameter IRC_VNORM0 .
