LIBMBD_PARALLEL_MODE = auto | kpoints | atoms Default: LIBMBD_PARALLEL_MODE =auto (default in libMBD) Description: LIBMBD_PARALLEL_MODE selects the parallelization scheme used in the library libMBD of many-body dispersion methods    . LIBMBD_PARALLEL_MODE allows to choose the parallelization scheme used in the library libMBD of many-body dispersion methods    . The value is internally passed to the libMBD input parallel_mode described at the page  . libMBD is a separate library package that has to be downloaded  and compiled before VASP is compiled with the corresponding precompiler options and links to the libraries .
