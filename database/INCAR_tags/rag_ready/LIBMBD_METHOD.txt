LIBMBD_METHOD =  Default: LIBMBD_METHOD = mbd-rsscs (default in libMBD) Description: LIBMBD_METHOD selects one of the methods available in the library libMBD of many-body dispersion methods    . LIBMBD_METHOD can be set to a label (string) corresponding to one of the methods listed on the libMBD website (see method at the page  ). libMBD is a separate library package that has to be downloaded  and compiled before VASP is compiled with the corresponding precompiler options and links to the libraries .
