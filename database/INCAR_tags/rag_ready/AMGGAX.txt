AMGGAX =  Description: AMGGAX is a parameter that multiplies the meta-GGA exchange functional (available as of VASP.6.4.0). AMGGAX can be used as the fraction of meta-GGA exchange in a Hartree-Fock/DFT hybrid functional (possible since VASP.6.4.0).
