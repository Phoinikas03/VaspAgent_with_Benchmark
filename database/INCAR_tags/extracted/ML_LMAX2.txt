<p>The tag <a class="mw-selflink selflink">ML_LMAX2</a> specifies the maximum angular momentum quantum number <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle L_\max}" xmlns="http://www.w3.org/1998/Math/MathML">

</p>
<p>By default an angular filtering (see <a href="/wiki/index.php/ML_LAFILT2" title="ML LAFILT2">ML_LAFILT2</a> and <a href="/wiki/index.php/ML_IAFILT2" title="ML IAFILT2">ML_IAFILT2</a>) with a filtering parameter of <a href="/wiki/index.php/ML_AFILT2" title="ML AFILT2">ML_AFILT2</a>=0.002 is used where a value of <a class="mw-selflink selflink">ML_LMAX2</a>=3 is perfectly safe for most applications. If you need to improve the accuracy, you might also try to increase it to <a class="mw-selflink selflink">ML_LMAX2</a>=4. If the angular filter is switched off, we recommend to use a value of <a class="mw-selflink selflink">ML_LMAX2</a>=6 (this is then the default value).
</p>
