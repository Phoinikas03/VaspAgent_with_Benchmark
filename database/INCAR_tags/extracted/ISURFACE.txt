</p>
<p>Description: <a class="mw-selflink selflink">KERNEL_TRUNCATION/ISURFACE</a> specifies the non-periodic dimension when performing calculations with 2D kernel truncation methods.
</p>
<p>When performing 2D kernel truncation with <a href="/wiki/index.php/KERNEL_TRUNCATION/IDIMENSIONALITY" title="KERNEL TRUNCATION/IDIMENSIONALITY">KERNEL_TRUNCATION/IDIMENSIONALITY</a>=2, <a class="mw-selflink selflink">KERNEL_TRUNCATION/ISURFACE</a> specifies which direction is non-periodic.
The most common use case is to set <a class="mw-selflink selflink">KERNEL_TRUNCATION/ISURFACE</a> to the dimension along the surface normal.
</p>
<p>If the surface normal points in the direction of the x-axis <a class="mw-selflink selflink">KERNEL_TRUNCATION/ISURFACE</a> = 1, if it is along the y-axis, <a class="mw-selflink selflink">KERNEL_TRUNCATION/ISURFACE</a> = 2 and if it is along the z-axis, <a class="mw-selflink selflink">KERNEL_TRUNCATION/ISURFACE</a> = 3.
</p>
