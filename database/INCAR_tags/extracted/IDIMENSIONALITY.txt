<p><a class="mw-selflink selflink">KERNEL_TRUNCATION/IDIMENSIONALITY</a> =  0 | 2 | 3<br/> Default: <b>KERNEL_TRUNCATION/IDIMENSIONALITY</b> =  3 
</p>
<p>Description: <a class="mw-selflink selflink">KERNEL_TRUNCATION/IDIMENSIONALITY</a> specifies the boundary condition used to compute the hartree and ionic potential. 
</p>
<p>If <a href="/wiki/index.php/KERNEL_TRUNCATION/LTRUNCATE" title="KERNEL TRUNCATION/LTRUNCATE">KERNEL_TRUNCATION/LTRUNCATE</a> = T, <a class="mw-selflink selflink">KERNEL_TRUNCATION/IDIMENSIONALITY</a> determines the boundary condition that is used to compute the local potential.
The default value of 3 implies that the system is periodic in all dimensions, i.e. there is no influence of kernel truncation on the resulting energies and forces.
Setting <a class="mw-selflink selflink">KERNEL_TRUNCATION/IDIMENSIONALITY</a> to either 0 or 2 uses the 0D and 2D truncated kernel respectively.<sup class="reference" id="cite_ref-vijay:arxiv:2024_1-0"><a href="#cite_note-vijay:arxiv:2024-1">[1]</a></sup><sup class="reference" id="cite_ref-rozzi:prb:2006_2-0"><a href="#cite_note-rozzi:prb:2006-2">[2]</a></sup><sup class="reference" id="cite_ref-sohier:prb:2017_3-0"><a href="#cite_note-sohier:prb:2017-3">[3]</a></sup>
These kernels create 0D (i.e. no periodic interactions, as is the case of molecules) and 2D (i.e. periodic interactions only in two dimensions, as in the case for surfaces).
</p>
