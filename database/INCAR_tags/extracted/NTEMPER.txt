<p><a class="mw-selflink selflink">NTEMPER</a> = [integer]<br/> Default: <b>NTEMPER</b> =  200 
</p>
<p>Description: <a class="mw-selflink selflink">NTEMPER</a> specifies how often temperature swaps are attempted during parallel tempering. The flag must be used in combination with  <a href="/wiki/index.php/IMAGES" title="IMAGES">IMAGES</a> and <a href="/wiki/index.php/LTEMPER" title="LTEMPER">LTEMPER</a>. <a class="mw-selflink selflink">NTEMPER</a>=0, allows to perform several completely independent calculations.
</p>
<p><a class="mw-selflink selflink">NTEMPER</a> must be a positive integer or zero. Swaps are on average attempted after <a class="mw-selflink selflink">NTEMPER</a> molecular dynamics steps.

Obviously, for <a class="mw-selflink selflink">NTEMPER</a>=1 swaps are attempted at every MD step. In the present code version, temperature swaps are alternatively performed between images 1,2 and 3,4 and 5,6 etc., or upon the next swap between images 2,3 and 4,5 and 6,7 etc.
This implies that many swaps are attempted at the same time.
</p>
<p>Furthermore, no temperature swapping is attempted for <a class="mw-selflink selflink">NTEMPER</a>=0. This allows many independent calculations to be performed in directories 01, 02, 03, ... which can be useful when performing many similar calculations for small systems or using machine-learned force fields on many-core machines (strong scaling is sometimes not particularly good for small systems).
</p>
<p>A final note is in place. If INCAR files exist in the subdirectories, the tag <a class="mw-selflink selflink">NTEMPER</a>  must be set in all <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> files of the subdirectories 01, 02, 03, and <a class="mw-selflink selflink">NTEMPER</a> must be identical. Failure to observe this rule can lead to unexpected behavior.
</p>
