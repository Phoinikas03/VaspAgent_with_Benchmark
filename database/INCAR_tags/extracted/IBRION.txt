<h2><span class="mw-headline" id="Molecular_dynamics">Molecular dynamics</span></h2>
<p>In <a href="/wiki/index.php/Category:Molecular_dynamics" title="Category:Molecular dynamics">molecular-dynamics (MD) simulations</a> the positions of the ions are updated using a classical equation of motion for the ions. There are several algorithms for the <a href="/wiki/index.php/Time-propagation_algorithms_in_molecular_dynamics" title="Time-propagation algorithms in molecular dynamics">time propagation in MD</a> controlled by selecting <a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a> and the choice of the <a class="mw-redirect" href="/wiki/index.php/Thermostats" title="Thermostats">thermostats</a>. The MD run performs <a href="/wiki/index.php/NSW" title="NSW">NSW</a> timesteps of length <a href="/wiki/index.php/POTIM" title="POTIM">POTIM</a>.
</p>
<p>Frequently, performing an <a class="mw-redirect" href="/wiki/index.php/Electronic_minimization" title="Electronic minimization">ab-initio calculations</a> in every step of an MD simulation is too expensive so that <a href="/wiki/index.php/Category:Machine-learned_force_fields" title="Category:Machine-learned force fields">machine-learned force fields</a> are needed.
</p>
<h2><span class="mw-headline" id="Structure_optimization">Structure optimization</span></h2>
<p>VASP optimizes the structure based on the degrees of freedom selected with the <a href="/wiki/index.php/ISIF" title="ISIF">ISIF</a> tag and (if used) the selective dynamics <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> file.
Generally, the larger the number of degrees of freedom, the harder it is to find the optimal solution.
To find the solution, VASP provides multiple algorithms:
</p>
<ul><li>RMM-DIIS (<a class="mw-selflink selflink">IBRION</a>=1) reduces the forces by linear combination of previous positions. It is the method of choice for larger systems (&gt;20 degrees of freedom) that are reasonably close to the ground-state structure.</li>
<li>Conjugate gradient (<a class="mw-selflink selflink">IBRION</a>=2) finds the optimal step size along a search direction. It is a robust default choice but may need more iterations than RMM-DIIS.</li>
<li>Damped molecular dynamics (<a class="mw-selflink selflink">IBRION</a>=3) runs a MD simulation with decreasing velocity of the ions. Use this for large systems far away from the minimum to get to a better starting point for the other algorithms.</li></ul>
<p>Consult the <a href="/wiki/index.php/Structure_optimization" title="Structure optimization">structure optimization</a> page for advise on how to choose the optimization algorithm.
</p>

</p>
<ul><li><code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 5</code> <a href="/wiki/index.php/Phonons_from_finite_differences" title="Phonons from finite differences">finite differences</a> <b>without</b> symmetry</li>
<li><code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 6</code> <a href="/wiki/index.php/Phonons_from_finite_differences" title="Phonons from finite differences">finite differences</a> <b>with</b> symmetry</li>
<li><code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 7</code> <a href="/wiki/index.php/Phonons_from_density-functional-perturbation_theory" title="Phonons from density-functional-perturbation theory">density-functional-perturbation theory</a> <b>without</b> symmetry</li>
<li><code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 8</code> <a href="/wiki/index.php/Phonons_from_density-functional-perturbation_theory" title="Phonons from density-functional-perturbation theory">density-functional-perturbation theory</a> <b>with</b> symmetry</li></ul>
<p>For finite differences, the elastic tensors and internal strain tensors is computed for <a href="/wiki/index.php/ISIF" title="ISIF">ISIF</a>&gt;=3.
Compute Born-effective charges, piezoelectric constants, and the ionic contribution to the dielectric tensor by specifying <code style="padding: 2px"><a href="/wiki/index.php/LEPSILON" title="LEPSILON">LEPSILON</a> = .TRUE.</code> (<a class="mw-redirect" href="/wiki/index.php/Linear_response" title="Linear response">linear response theory</a>) or <code style="padding: 2px"><a href="/wiki/index.php/LCALCEPS" title="LCALCEPS">LCALCEPS</a> = .TRUE.</code> (finite external field).
</p>

<h2><span class="mw-headline" id="Analyzing_transition_states">Analyzing transition states</span></h2>
<p>To study the kinetics of chemical reactions, one may want to construct <a class="mw-redirect" href="/wiki/index.php/Transition_states" title="Transition states">transition states</a> or follow the reaction path.
For the analysis of transition states the following methods are available:
</p>
<ul><li>Setting <code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 40</code>, you can start from a transition state and monitor the energy along an intrinsic-reaction coordinate (IRC). The <a class="mw-redirect" href="/wiki/index.php/IRC_calculations" title="IRC calculations">IRC calculations</a> section describes this method.</li>
<li>With the <a href="/wiki/index.php/Improved_dimer_method" title="Improved dimer method">improved dimer method</a> (<code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 44</code>), you can search for a the transition state starting from an arbitrary structure in the investigated phase space.</li>
<li>The <a href="/wiki/index.php/Nudged_elastic_bands" title="Nudged elastic bands">nudged elastic bands</a> method finds an approximate reaction path based on the initial and final structure, i.e., reactant and product.</li></ul>
<h2><span class="mw-headline" id="Interactively_supplied_positions_and_lattice_vectors">Interactively supplied positions and lattice vectors</span></h2>
<p>Occasionally, you may want to run VASP for related structures where the overhead of restarting VASP is significant.
In these scenarios, VASP provides the following alternatives
</p>
<ul><li>With <code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 11</code>, you can provide new structures via the standard input. For <a href="/wiki/index.php/ISIF" title="ISIF">ISIF</a>&gt;=3, a complete <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> file is read, otherwise just the positions in fractional coordinates.</li></ul>
<ul><li>If you <a href="/wiki/index.php/Makefile.include#Plugins_(optional)" title="Makefile.include">linked VASP with Python</a>, you can <a href="/wiki/index.php/Plugins" title="Plugins">write a Python plugin</a> to modify the structure. Set <code style="padding: 2px"><a class="mw-selflink selflink">IBRION</a> = 12</code> or <code style="padding: 2px"><a href="/wiki/index.php/PLUGINS/STRUCTURE" title="PLUGINS/STRUCTURE">PLUGINS/STRUCTURE</a> = T</code> to activate it.</li></ul>
