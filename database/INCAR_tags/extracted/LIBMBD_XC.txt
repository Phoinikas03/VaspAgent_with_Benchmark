<p>Default: The functional set by the <a href="/wiki/index.php/GGA" title="GGA">GGA</a>, <a href="/wiki/index.php/METAGGA" title="METAGGA">METAGGA</a> or <a href="/wiki/index.php/XC" title="XC">XC</a> tag
</p>
<p>Description: <a class="mw-selflink selflink">LIBMBD_XC</a> sets the exchange-correlation functional for the setting of damping parameters used in the methods available in the library libMBD of many-body dispersion methods<sup class="reference" id="cite_ref-libmbd_1_1-0"><a href="#cite_note-libmbd_1-1">[1]</a></sup><sup class="reference" id="cite_ref-libmbd_2_2-0"><a href="#cite_note-libmbd_2-2">[2]</a></sup><sup class="reference" id="cite_ref-hermann:jcp:2023_3-0"><a href="#cite_note-hermann:jcp:2023-3">[3]</a></sup>.
</p>
<p><a class="mw-selflink selflink">LIBMBD_XC</a> allows to choose the exchange-correlation functional that determines which set of damping parameters is used in the methods available in the library libMBD of many-body dispersion methods. The value is internally passed to the libMBD input <b>xc</b> described at the page <sup class="reference" id="cite_ref-libmbd_input_4-0"><a href="#cite_note-libmbd_input-4">[4]</a></sup>.
</p>
<p>The possible choices depend on the dispersion method selected with the <a href="/wiki/index.php/LIBMBD_METHOD" title="LIBMBD METHOD">LIBMBD_METHOD</a> tag and are listed in the file mbd_damping.F90 of the libMBD source code. If <a class="mw-selflink selflink">LIBMBD_XC</a>=none is chosen, then no set of damping parameters is selected and either <a href="/wiki/index.php/LIBMBD_TS_SR" title="LIBMBD TS SR">LIBMBD_TS_SR</a> or <a href="/wiki/index.php/LIBMBD_MBD_BETA" title="LIBMBD MBD BETA">LIBMBD_MBD_BETA</a> has to be set.
</p>
<p>libMBD is a separate library package that has to be downloaded<sup class="reference" id="cite_ref-libmbd_2_2-1"><a href="#cite_note-libmbd_2-2">[2]</a></sup> and compiled before VASP is compiled with the corresponding <a href="/wiki/index.php/Precompiler_options#-DLIBMBD" title="Precompiler options">precompiler options</a> and <a href="/wiki/index.php/Makefile.include#LIBMBD_.28optional.29" title="Makefile.include">links to the libraries</a>.
</p>
