<p><a class="mw-selflink selflink">ML_CDOUB</a> = [real] 
</p>
<p>Description: This tag controls the criterion for "enforced" DFT calculations within the machine learning force field method.
</p>
<p>The usage of this tag in combination with the learning algorithms is described here: <a href="/wiki/index.php/Machine_learning_force_field_calculations:_Basics#Sampling_of_training_data_and_local_reference_configurations" title="Machine learning force field calculations: Basics">here</a>.
</p>
<p>If at any time, the estimated force errors are <a class="mw-selflink selflink">ML_CDOUB</a> times larger than the Bayesian threshold (i.e. "critically" high), a first principles calculation is performed and a new force field is immediately generated (even if the counter for sampling is below the minimum amount of sampled structures <a href="/wiki/index.php/ML_NMDINT" title="ML NMDINT">ML_NMDINT</a>).
</p>
