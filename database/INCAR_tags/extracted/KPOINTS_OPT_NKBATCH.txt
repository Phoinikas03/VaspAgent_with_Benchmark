<p><a class="mw-selflink selflink">KPOINTS_OPT_NKBATCH</a> = [integer]
</p>
<p>Default: <a class="mw-selflink selflink">KPOINTS_OPT_NKBATCH</a> = Number of k-points in the irreducible Brillouin zone of the self-consistent calculation. 
</p>
<p>Description: <a class="mw-selflink selflink">KPOINTS_OPT_NKBATCH</a> determines the size of the batch of k-points for the KPOINTS_OPT driver.
</p>
<p>When the <a href="/wiki/index.php/KPOINTS_OPT" title="KPOINTS OPT">KPOINTS_OPT</a> is present an additional non-self-consistent calculation is performed after self-consistency is reached.
This one-shot calculation is done in batches of N k-points to reduce memory usage.
Increasing the size of the batch leads to faster calculation times but higher memory usage in the non-self-consistent calculation.
</p>
