<p><a class="mw-selflink selflink">LANGEVIN_GAMMA_L</a> = [Real]<br/> Default: <b>LANGEVIN_GAMMA_L</b> = 0 
</p>
<p>Description: <a class="mw-selflink selflink">LANGEVIN_GAMMA_L</a> specifies the friction coefficient (in ps<sup>-1</sup>) for lattice degrees-of-freedom in case of Parrinello-Rahman dynamics (in case VASP was compiled with <a class="mw-redirect" href="/wiki/index.php/Precompiler_flags" title="Precompiler flags">-Dtbdyn</a>).
</p>
<p>When running <i>NpT</i> simulations with a <a href="/wiki/index.php/MDALGO#LangevinEOM" title="MDALGO">Langevin thermostat</a><sup class="reference" id="cite_ref-Allen91_1-0"><a href="#cite_note-Allen91-1">[1]</a></sup> (<a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a>=3), using the method of <a href="/wiki/index.php/MDALGO#ParrinelloRahman" title="MDALGO">Parrinello and Rahman</a>,<sup class="reference" id="cite_ref-Parrinello80_2-0"><a href="#cite_note-Parrinello80-2">[2]</a></sup><sup class="reference" id="cite_ref-Parrinello81_3-0"><a href="#cite_note-Parrinello81-3">[3]</a></sup> the friction coefficient for lattice degrees-of-freedom have to be specified (in ps<sup>-1</sup>) by means of the <a class="mw-selflink selflink">LANGEVIN_GAMMA_L</a>-tag.
A fictitious mass for the lattice degrees-of-freedom has to be assigned using the <a href="/wiki/index.php/PMASS" title="PMASS">PMASS</a> tag.
</p>
<p>The friction coefficients γ for the atomic degrees-of-freedom are specified using the <a href="/wiki/index.php/LANGEVIN_GAMMA" title="LANGEVIN GAMMA">LANGEVIN_GAMMA</a>-tag.
</p>
