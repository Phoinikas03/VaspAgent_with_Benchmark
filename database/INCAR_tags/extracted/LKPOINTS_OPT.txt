<p><a class="mw-selflink selflink">LKPOINTS_OPT</a> = .TRUE.| .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LKPOINTS_OPT</a> controlls whether VASP reads the <a href="/wiki/index.php/KPOINTS_OPT" title="KPOINTS OPT">KPOINTS_OPT</a> file.
</p>
<p>To avoid reading the <a href="/wiki/index.php/KPOINTS_OPT" title="KPOINTS OPT">KPOINTS_OPT</a> file without removing it from the working directory, the <a class="mw-selflink selflink">LKPOINTS_OPT</a> tag can be set to <code>.FALSE.</code> in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file.
</p>
