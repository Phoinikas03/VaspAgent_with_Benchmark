Generally, <a class="mw-selflink selflink">LSPECTRALGW</a> affects the compute time very little. QP energies also hardly
change when <a class="mw-selflink selflink">LSPECTRALGW</a>  is modified.
However, <a class="mw-selflink selflink">LSPECTRALGW</a> = .TRUE. is usually slightly more robust,
and should be selected for molecules and other systems with flat dispersion-less bands.  
One the other hand, <a class="mw-selflink selflink">LSPECTRALGW</a> = .TRUE. seems to converge slightly slower, 
as the complex shift <a href="/wiki/index.php/CSHIFT" title="CSHIFT">CSHIFT</a> is decreased. Set this flag, if the QP energies
show erratic behavior, for instance, if QP energies or Z-factors are not in the expected
range of values (0.5&lt;Z&lt;0.9).
