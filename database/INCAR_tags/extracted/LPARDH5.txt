<p><a class="mw-selflink selflink">LPARDH5</a> = [logical]<br/> Default: <b>LPARDH5</b> = .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LPARDH5</a> determines whether the partial charges are written to <a href="/wiki/index.php/PARCHG" title="PARCHG">PARCHG</a> or <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a>.
</p>
<p><a class="mw-redirect" href="/wiki/index.php/Band-decomposed_charge_densities" title="Band-decomposed charge densities">Partial charges</a> can be calculated as a postprocessing step when <a href="/wiki/index.php/LPARD" title="LPARD">LPARD</a> = .TRUE.. The output is written to one or several <a href="/wiki/index.php/PARCHG" title="PARCHG">PARCHG</a> files if <a class="mw-selflink selflink">LPARDH5</a>=.FALSE., and to <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a> if <a class="mw-selflink selflink">LPARDH5</a>=.TRUE.. If <a href="/wiki/index.php/NBMOD" title="NBMOD">NBMOD</a> = -1, the setting of <a href="/wiki/index.php/LPARD" title="LPARD">LPARD</a> is irrelevant. Instead of a <a href="/wiki/index.php/PARCHG" title="PARCHG">PARCHG</a> file, or a partial_charges group in the <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a> hdf5 file, a <a href="/wiki/index.php/CHGCAR" title="CHGCAR">CHGCAR</a> file <i>without</i> augmentation charges will be written.
</p>
<p>If the output is redirected to <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a>, <a class="external text" href="https://vasp.at/py4vasp/latest/index.html" rel="nofollow">py4vasp</a> can be used to analyze the partial charge density and to simulate STM pictures.
</p>
