<p><a class="mw-selflink selflink">WRT_POTENTIAL</a> = string<br/> Default: <b>WRT_POTENTIAL</b> = None 
</p>
<p>Description: Select which component of the local potential to be written as a post-processing step.
</p>
<p><a class="mw-selflink selflink">WRT_POTENTIAL</a> can select one or multiple local potentials on the real-space grid in the unit cell to be written, e.g.,
</p>
<p>or
</p>
<p>The output is written to <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a> and can be accessed either by <a class="external text" href="https://vasp.at/py4vasp/latest/index.html" rel="nofollow">py4vasp</a> or HDF5 command-line tools (h5ls, h5dump).
</p>
<p>The above allows the creation of a Python dictionary with the potential data.
</p>
<p>The above shows the table of contents of <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a>. Depending on the keywords specified with <a class="mw-selflink selflink">WRT_POTENTIAL</a> and the system it yields
</p>
<p>The grid density can be increased by choosing a higher value for <a href="/wiki/index.php/ENCUT" title="ENCUT">ENCUT</a> or explicitly by <a href="/wiki/index.php/NGX" title="NGX">NGX</a>, <a href="/wiki/index.php/NGY" title="NGY">NGY</a>, <a href="/wiki/index.php/NGZ" title="NGZ">NGZ</a>.
</p>
<p>The first dimension of the datasets in /results/potential is 1 for nonmagnetic calculation, 2 for spin-polarized calculation, and 4 for noncollinear calculations. In case the potential is scalar, i.e., has no B-field-like contribution that couples to the magnetization, only the 1st component exists. Hence, for <i>hartree</i> and <i>ionic</i>, the first dimension is 1.
