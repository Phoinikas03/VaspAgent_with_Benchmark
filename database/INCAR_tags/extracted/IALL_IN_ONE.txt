<p>Description: <a class="mw-selflink selflink">IALL_IN_ONE</a>&gt;0 enables the all-in-one mode for <a class="mw-redirect" href="/wiki/index.php/Many-body_perturbation_theory" title="Many-body perturbation theory">many-body perturbation theory</a> calculations,
i.e.,  <a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a>=<a href="/wiki/index.php/ACFDT/RPA_calculations" title="ACFDT/RPA calculations">ACFDT[R]</a>, <a href="/wiki/index.php/Practical_guide_to_GW_calculations" title="Practical guide to GW calculations">[EV]GW0[R]</a>, <a href="/wiki/index.php/Practical_guide_to_GW_calculations#Fully_self-consistent_GW_caluclations:_GWR" title="Practical guide to GW calculations">GWR</a>. 
</p>
<p>In the all-in-one mode, VASP automatically performs the necessary DFT steps prior to the many-body perturbation theory (MBPT) calculation, i.e. a DFT calculation with <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a>, followed by an exact diagonalization of the Kohn-Sham Hamiltonian with <a href="/wiki/index.php/NBANDSEXACT" title="NBANDSEXACT">NBANDSEXACT</a> bands. 
Note, <a href="/wiki/index.php/NBANDSEXACT" title="NBANDSEXACT">NBANDSEXACT</a> is set by default to the maximum number of plane-waves given by the chosen energy cutoff for the orbitals <a href="/wiki/index.php/ENCUT" title="ENCUT">ENCUT</a>. 
In the all-in-one mode, the actual GW/RPA calculation is also performed with <a href="/wiki/index.php/NBANDSEXACT" title="NBANDSEXACT">NBANDSEXACT</a> bands. 
If <a href="/wiki/index.php/NBANDS_WAVE" title="NBANDS WAVE">NBANDS_WAVE</a> is not set, all orbitals are written to <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a>, which potentially becomes huge in file size.
</p>
<p>The all-in-one mode is automatically enabled for <a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a>=[EV]GW[0]R, RPA[R],ACFDT[R] if <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a> is not set.
</p>
