<p><a class="mw-selflink selflink">LAECHG</a> = .TRUE. | .FALSE.<br/> Default: <b>LAECHG</b> = .FALSE. 
</p>
<p>Description: when <a class="mw-selflink selflink">LAECHG</a>=.TRUE. the <i>all-electron</i> charge density will be reconstructed explicitly and written to files.
</p>
<p>If <a class="mw-selflink selflink">LAECHG</a>=.TRUE. is set VASP will reconstruct the <i>all-electron</i> charge density on the so-called "fine" FFT-grid.
This "fine" FFT-grid consists of <a href="/wiki/index.php/NGXF" title="NGXF">NGXF</a>×<a href="/wiki/index.php/NGYF" title="NGYF">NGYF</a>×<a href="/wiki/index.php/NGZF" title="NGZF">NGZF</a> points in real space (<i>i.e.</i>, the grid that is used to represent the augmented pseudo charge densities of the USPP and PAW methods).
</p>
<p>In fact, for <a class="mw-selflink selflink">LAECHG</a>=.TRUE., VASP will reconstruct three distinct <i>all-electron</i> densities:
</p>
<ol><li>the core density.</li>
<li>the proto-atomic valence density (overlapping atomic charge densities).</li>
<li>the self-consistent valence density.</li></ol>
<p>These are written to the files <a class="new" href="/wiki/index.php?title=AECCAR0&amp;action=edit&amp;redlink=1" title="AECCAR0 (page does not exist)">AECCAR0</a>, <a class="new" href="/wiki/index.php?title=AECCAR1&amp;action=edit&amp;redlink=1" title="AECCAR1 (page does not exist)">AECCAR1</a>, and <a class="new" href="/wiki/index.php?title=AECCAR2&amp;action=edit&amp;redlink=1" title="AECCAR2 (page does not exist)">AECCAR2</a>, respectively.
The first two of these files are written at the start of the run, whereas the last is written at the end after self-consistency has been reached.
</p>
<p><b>N.B.:</b> In the language of the <a href="/wiki/index.php/Projector-augmented-wave_formalism" title="Projector-augmented-wave formalism">PAW method</a> an "all-electron" density does <b>not</b> refer to the <i>density of all electrons</i>, instead it denotes a density that includes all the nodal features near the nucleus associated with the <i>true</i> (as opposed to the <i>pseudized</i>) one-electron orbitals.
