<p><a class="mw-selflink selflink">NBLK</a> = [integer] 
</p>
<p>Description: <a class="mw-selflink selflink">NBLK</a> determines the blocking factor in many BLAS level 3 routines.
</p>
<p>In some cases, VASP has to perform a unitary transformation of the current orbitals. This is done using a work array CBLOCK and the following FORTRAN code:
</p>

The task performed by this call is indicated by the comment line written above the ZGEMM call. Generally <a class="mw-selflink selflink">NBLK</a>=16 or 32 is large enough for super-scalar machines.
