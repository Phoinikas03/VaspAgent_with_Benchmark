<p>In VASP the eigenvalue spectrum of the charge dielectric matrix is calculated and written to the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a> file at each electronic step. This allows a rather easy optimization of the mixing parameters, if required. Search in the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a> file for
</p>
<p>The parameters for the mixing are optimal if the mean eigenvalue Γ<sub>mean</sub>=1, and if the width of the eigenvalue spectrum is minimal. For an initial linear mixing (<a class="mw-selflink selflink">BMIX</a>≈0) an optimal setting for <a href="/wiki/index.php/AMIX" title="AMIX">AMIX</a> can be found easily by setting <a href="/wiki/index.php/AMIX" title="AMIX">AMIX</a><sub>optimal</sub>=<a href="/wiki/index.php/AMIX" title="AMIX">AMIX</a><sub>current</sub>*Γ<sub>mean</sub>. For the Kerker scheme (<a href="/wiki/index.php/IMIX" title="IMIX">IMIX</a>=1) either <a href="/wiki/index.php/AMIX" title="AMIX">AMIX</a> or <a class="mw-selflink selflink">BMIX</a> can be optimized, but we recommend to change only <a class="mw-selflink selflink">BMIX</a> and keep <a href="/wiki/index.php/AMIX" title="AMIX">AMIX</a> fixed (you must decrease <a class="mw-selflink selflink">BMIX</a> if the mean eigenvalue is larger than one, and increase <a class="mw-selflink selflink">BMIX</a> if the mean eigenvalue Γ<sub>mean</sub>&lt;1).
</p>
