</math></span><img alt="{\displaystyle d}" aria-hidden="true" class="mwe-math-fallback-image-inline" src="/wiki/index.php?title=Special:MathShowImage&amp;hash=8277e0910d750195b448797616e091ad&amp;mode=mathml" style="vertical-align: -0.338ex; height: 2.176ex; width: 1.216ex;"/></span> in the Tkatchenko-Scheffler method<sup class="reference" id="cite_ref-tkatchenko:prl:09_1-0"><a href="#cite_note-tkatchenko:prl:09-1">[1]</a></sup> as implemented in the library libMBD of many-body dispersion methods<sup class="reference" id="cite_ref-libmbd_1_2-0"><a href="#cite_note-libmbd_1-2">[2]</a></sup><sup class="reference" id="cite_ref-libmbd_2_3-0"><a href="#cite_note-libmbd_2-3">[3]</a></sup><sup class="reference" id="cite_ref-hermann:jcp:2023_4-0"><a href="#cite_note-hermann:jcp:2023-4">[4]</a></sup>.
</p>
<p><a class="mw-selflink selflink">LIBMBD_TS_D</a> allows to choose the value of the damping parameter <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle d}" xmlns="http://www.w3.org/1998/Math/MathML">

<mrow class="MJX-TeXAtom-ORD">
<mstyle displaystyle="true" scriptlevel="0">
<mi>d</mi>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle d}</annotation>
</semantics>
</math></span><img alt="{\displaystyle d}" aria-hidden="true" class="mwe-math-fallback-image-inline" src="/wiki/index.php?title=Special:MathShowImage&amp;hash=8277e0910d750195b448797616e091ad&amp;mode=mathml" style="vertical-align: -0.338ex; height: 2.176ex; width: 1.216ex;"/></span> in the Tkatchenko-Scheffler method<sup class="reference" id="cite_ref-tkatchenko:prl:09_1-1"><a href="#cite_note-tkatchenko:prl:09-1">[1]</a></sup> as implemented in the library libMBD of many-body dispersion methods<sup class="reference" id="cite_ref-libmbd_1_2-1"><a href="#cite_note-libmbd_1-2">[2]</a></sup><sup class="reference" id="cite_ref-libmbd_2_3-1"><a href="#cite_note-libmbd_2-3">[3]</a></sup><sup class="reference" id="cite_ref-hermann:jcp:2023_4-1"><a href="#cite_note-hermann:jcp:2023-4">[4]</a></sup>. The value is internally passed to the libMBD input <b>ts_d</b> described at the page <sup class="reference" id="cite_ref-libmbd_input_5-0"><a href="#cite_note-libmbd_input-5">[5]</a></sup>. <a class="mw-selflink selflink">LIBMBD_TS_D</a> is the same as the <a href="/wiki/index.php/VDW_D" title="VDW D">VDW_D</a> tag that is used for the VASP implementation of the Tkatchenko-Scheffler method.
</p>
<p>libMBD is a separate library package that has to be downloaded<sup class="reference" id="cite_ref-libmbd_2_3-2"><a href="#cite_note-libmbd_2-3">[3]</a></sup> and compiled before VASP is compiled with the corresponding <a href="/wiki/index.php/Precompiler_options#-DLIBMBD" title="Precompiler options">precompiler options</a> and <a href="/wiki/index.php/Makefile.include#LIBMBD_.28optional.29" title="Makefile.include">links to the libraries</a>.
</p>
