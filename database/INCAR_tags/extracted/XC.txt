<p><a class="mw-selflink selflink">XC</a> = Combination of functionals 
</p>
<p>Description: Specifies a combination of exchange-correlation functionals.
</p>
<p>A combination of semilocal (LDA, GGA, and METAGGA) functionals can be set with the <a class="mw-selflink selflink">XC</a> tag, which provides much more flexibility in the choice of the functional compared to the <a href="/wiki/index.php/GGA" title="GGA">GGA</a> and <a href="/wiki/index.php/METAGGA" title="METAGGA">METAGGA</a> tags. The functionals that can be combined are the functionals implemented in VASP (listed at <a href="/wiki/index.php/GGA" title="GGA">GGA</a> and <a href="/wiki/index.php/METAGGA" title="METAGGA">METAGGA</a>) and the functionals implemented in Libxc<sup class="reference" id="cite_ref-marques:cpc:2012_1-0"><a href="#cite_note-marques:cpc:2012-1">[1]</a></sup><sup class="reference" id="cite_ref-lehtola:sx:2018_2-0"><a href="#cite_note-lehtola:sx:2018-2">[2]</a></sup><sup class="reference" id="cite_ref-libxc_3-0"><a href="#cite_note-libxc-3">[3]</a></sup> (listed on the Libxc website<sup class="reference" id="cite_ref-libxc_list_4-0"><a href="#cite_note-libxc_list-4">[4]</a></sup>). The combination can consist of up to 100 components; for each, a multiplication factor can be set with the <a href="/wiki/index.php/XC_C" title="XC C">XC_C</a> tag.
</p>
