<p><a class="mw-selflink selflink">PHON_TLIST</a> = [real array]<br/> Default: <b>PHON_TLIST</b> = none 
</p>
<p>Description: This flag provides the list of temperatures for that the electron-phonon interaction using the ZG configuration<sup class="reference" id="cite_ref-zacharias:prb:2016_1-0"><a href="#cite_note-zacharias:prb:2016-1">[1]</a></sup> is evaluated. 
</p>
<p>This feature is only available from VASP version 6.0 or higher.
</p>
<p>Also, the number of temperatures defined in <a href="/wiki/index.php/PHON_NTLIST" title="PHON NTLIST">PHON_NTLIST</a> has to be defined together with this tag.
</p>
<p>For further usage of this tag see: <a href="/wiki/index.php/Electron-phonon_interactions_from_Monte-Carlo_sampling" title="Electron-phonon interactions from Monte-Carlo sampling">Electron-phonon interactions from Monte-Carlo sampling</a>.
</p>
