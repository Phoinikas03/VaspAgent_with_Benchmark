<p><a class="mw-selflink selflink">LTRIPLET</a> = .TRUE. | .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LTRIPLET</a> selects a triplet ansatz for <a href="/wiki/index.php/Category:Bethe-Salpeter_equations" title="Category:Bethe-Salpeter equations">Bethe-Salpeter-equations</a> (BSE) calculations. This is synonymous to  <a href="/wiki/index.php/LHARTREE" title="LHARTREE">LHARTREE</a> = .FALSE.
</p>
<p>For a usual BSE calculation (<a href="/wiki/index.php/LHARTREE" title="LHARTREE">LHARTREE</a>=.TRUE.; <a class="mw-selflink selflink">LTRIPLET</a> = .FALSE.), the excited state corresponds to a singlet if the ground state is not spin-polarized (<a href="/wiki/index.php/ISPIN" title="ISPIN">ISPIN</a>=1) or anti-ferromagnetic (<a href="/wiki/index.php/ISPIN" title="ISPIN">ISPIN</a>=2, total magnetic moment 0). This is so because the ansatz for the BSE calculation involves a hole and an electron pair prepared within each spin channel. For instance, an electron with up spin is removed from the ground-state determinant and placed with the same spin into a previously unoccupied orbital. If there are no separate spin channels this ansatz results in a net spin-zero, which corresponds to a singlet state.
</p>
<p>If <a class="mw-selflink selflink">LTRIPLET</a>=.TRUE., VASP assumes that an <i>up</i> electron is removed from the ground-state determinant and placed as a <i>down</i> electron into a previously unoccupied orbital (spin flip). This ansatz corresponds to a triplet state as it has a net spin of one. Without spin-orbit coupling (<a href="/wiki/index.php/LSORBIT" title="LSORBIT">LSORBIT</a> = .FALSE.), the transition probability from the singlet ground state to the triplet solutions will be exactly zero. These excitations correspond to dark, potentially long-lived triplet excitons. However, VASP calculates the transition probabilities incorrectly, by assuming that a spin-flip excitation has the same transition probabilities as a usual singlet excitation.
The reported transition probabilities ("optical transitions" in <a class="new" href="/wiki/index.php?title=Vasprun.xml&amp;action=edit&amp;redlink=1" title="Vasprun.xml (page does not exist)">vasprun.xml</a>) are hence incorrect; they should be all zero. Likewise, the contributions to the dielectric function are zero for triplet excitations,
and incorrectly reported in the <a class="new" href="/wiki/index.php?title=Vasprun.xml&amp;action=edit&amp;redlink=1" title="Vasprun.xml (page does not exist)">vasprun.xml</a> file.
</p>

<p>To obtain meaningful transition probabilities for the singlet and triplet excitations, include spin-orbit coupling by setting <a href="/wiki/index.php/LSORBIT" title="LSORBIT">LSORBIT</a> = .TRUE. in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file throughout the calculation, i.e., for the ground-state and BSE calculation.  In this case, the singlet and all three sets of triplet excitations are calculated simultaneously, and proper transition probabilities are assigned to each transition. Small transition probabilities might be observed even for triplet excitations as a result of spin-orbit coupling. <a class="mw-selflink selflink">LTRIPLET</a> = .TRUE. should not be used for calculations including spin-orbit coupling.
</p>
