<p><a class="mw-selflink selflink">PHON_NTLIST</a> = [integer]<br/> Default: <b>PHON_NTLIST</b> = none 
</p>
<p>Description: This flag sets the number of temperatures for that the electron-phonon interactions using the ZG configuration<sup class="reference" id="cite_ref-zacharias:prb:2016_1-0"><a href="#cite_note-zacharias:prb:2016-1">[1]</a></sup> is evaluated. 
</p>
<p>This feature is only available from VASP version 6.0 or higher.
</p>
<p>Also, a list of temperatures defined in <a href="/wiki/index.php/PHON_TLIST" title="PHON TLIST">PHON_TLIST</a> has to be defined together with this tag
</p>
<p>For further usage of this tag see: <a href="/wiki/index.php/Electron-phonon_interactions_from_Monte-Carlo_sampling" title="Electron-phonon interactions from Monte-Carlo sampling">Electron-phonon interactions from Monte-Carlo sampling</a>.
</p>
