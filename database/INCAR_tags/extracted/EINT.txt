<p><a class="mw-selflink selflink">EINT</a> =  [real1] [real2] | [real1]<br/> Default: <b>EINT</b> =  not set 
</p>
<p>Description: <a class="mw-selflink selflink">EINT</a> sets the energy interval for bands contributing to the calculation of the <a class="mw-redirect" href="/wiki/index.php/Band-decomposed_charge_densities" title="Band-decomposed charge densities">partial charge density</a> in eV.
</p>
<ul><li><a class="mw-selflink selflink">EINT</a>= [real1] [real2]:</li></ul>
<ul><li><a class="mw-selflink selflink">EINT</a>= [real1]:</li></ul>
<p>If [real1] is larger than [real2], the two values will be flipped internally, so a meaningful energy interval is used.
</p>
<p>If <a class="mw-selflink selflink">EINT</a> is set, but <a href="/wiki/index.php/NBMOD" title="NBMOD">NBMOD</a> is not, it will be internally set to <a href="/wiki/index.php/NBMOD" title="NBMOD">NBMOD</a> = -2, and the input values of <a class="mw-selflink selflink">EINT</a> will be treated as absolute energies.
</p>
<p><a class="mw-selflink selflink">EINT</a> can be conveniently used in combination with <a href="/wiki/index.php/NBMOD" title="NBMOD">NBMOD</a> = -3 to mimic the bias-voltage for <a href="/wiki/index.php/STM_of_graphene" title="STM of graphene">simulating a scanning-tunneling-microscope image</a>.
</p>
