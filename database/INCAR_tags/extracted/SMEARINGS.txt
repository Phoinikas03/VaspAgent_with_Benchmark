<p><a class="mw-selflink selflink">SMEARINGS</a> = [real array of length (2 * <a href="/wiki/index.php/NSW" title="NSW">NSW</a>)]<br/> Default: <b>SMEARINGS</b> = not set 
</p>
<p>Description: <a class="mw-selflink selflink">SMEARINGS</a> defines the smearing parameters for <code style="padding: 2px"><a href="/wiki/index.php/ISMEAR" title="ISMEAR">ISMEAR</a> = -3</code> in the calculation of the partial occupancies.
</p>
<p><code style="padding: 2px"><a href="/wiki/index.php/ISMEAR" title="ISMEAR">ISMEAR</a> = -3</code> performs a loop over smearing-parameters supplied in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file. With the tag <a class="mw-selflink selflink">SMEARINGS</a>, you select which smearings are used
</p>
<p>VASP will then read the provided smearings and conduct (<a href="/wiki/index.php/NSW" title="NSW">NSW</a> + 1) calculations with the different smearings.
For the first calculation, VASP uses tetrahedron smearing <code style="padding: 2px"><a href="/wiki/index.php/ISMEAR" title="ISMEAR">ISMEAR</a> = -5</code> to ensure that the tetrahedron information is present in case any of the selected smearings uses a tetrahedron method.
Since VASP uses the relaxation engine to loop over the different smearings you cannot combine <a class="mw-selflink selflink">SMEARINGS</a> with other relaxation methods <a href="/wiki/index.php/IBRION" title="IBRION">IBRION</a>.
</p>
