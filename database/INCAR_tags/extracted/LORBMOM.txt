<p><a class="mw-selflink selflink">LORBMOM</a> = .TRUE. | .FALSE.<br/> Default: <b>LORBMOM</b> = .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LORBMOM</a> specifies whether the orbital moments are written out or not (in a calculation using <a href="/wiki/index.php/LSORBIT" title="LSORBIT">LSORBIT</a>=.TRUE.).
</p>
<p>If <a class="mw-selflink selflink">LORBMOM</a>=.TRUE. is set VASP will  use the projectors of the PAW potentials to calculate the
orbital moment within the PAW spheres, and write them to the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a> file.
</p>
