<p>Description: <a class="mw-selflink selflink">LSPECTRAL</a> specifies to use the spectral method.

The downside of the coin is that the response function must be kept in memory for all considered frequencies, which can cause excessive memory requirements. VASP, therefore, distributes the dielectric functions among the available compute nodes.
</p>
<p>A similar trick is used when the QP-shifts are calculated. In general it is strongly recommended to set <a class="mw-selflink selflink">LSPECTRAL</a> = .TRUE., except if memory requirements are too excessive.
