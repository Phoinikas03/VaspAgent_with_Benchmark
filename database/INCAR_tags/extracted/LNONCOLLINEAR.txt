<p><a class="mw-selflink selflink">LNONCOLLINEAR</a> = .True. | .False. 
</p>
<p>Description: Switch on noncollinear magnetic calculations.
</p>
<p>For noncollinear magnetic calculations, set <a class="mw-selflink selflink">LNONCOLLINEAR</a> = True in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file and use the <code>vasp_ncl</code> executable. The electronic minimization treats the full 2x2 spin density<sup class="reference" id="cite_ref-hobbs:prb:00_1-0"><a href="#cite_note-hobbs:prb:00-1">[1]</a></sup>
</p>
<p>which is written to the <a href="/wiki/index.php/CHGCAR" title="CHGCAR">CHGCAR</a> file. In spinor space, the part of the spin density proportional to the 2x2 unit matrix corresponds to the charge density, and the part proportional to the vector of Pauli matrices is the magnetization density.
This enables the consideration of noncollinear magnetic structures within spin-density-functional theory. <a href="/wiki/index.php/MAGMOM" title="MAGMOM">MAGMOM</a> sets the initial magnetic moments. Write the final magnetic moments by setting <a href="/wiki/index.php/LORBIT" title="LORBIT">LORBIT</a>. 
</p>
<p>It is possible to <b>restart a noncollinear calculation</b> from a previous nonmagnetic calculation (<a href="/wiki/index.php/ISPIN" title="ISPIN">ISPIN</a>=1 and <a class="mw-selflink selflink">LNONCOLLINEAR</a>=F) or spin-polarized calculation (<a href="/wiki/index.php/ISPIN" title="ISPIN">ISPIN</a>=2) by reading <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> or <a href="/wiki/index.php/CHGCAR" title="CHGCAR">CHGCAR</a> files. The magnetization of the spin-polarized calculation is interpreted to point along <a href="/wiki/index.php/SAXIS" title="SAXIS">SAXIS</a> (default: Cartesian direction <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle \hat z}" xmlns="http://www.w3.org/1998/Math/MathML">

<mover>
<mi>z</mi>

</mover>
</mrow>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle \hat z}</annotation>
</semantics>
</math></span><img alt="{\displaystyle \hat z}" aria-hidden="true" class="mwe-math-fallback-image-inline" src="/wiki/index.php?title=Special:MathShowImage&amp;hash=8525852f0aeec3b7f461623645f3e4a8&amp;mode=mathml" style="vertical-align: -0.338ex; height: 2.176ex; width: 1.296ex;"/></span>). It is not possible to rotate the magnetic moment locally on selected atoms when restarting with a magnetization density. The magnetic configuration can globally be rotated with respect to the lattice by restarting with a different <a href="/wiki/index.php/SAXIS" title="SAXIS">SAXIS</a>.
</p>
<p>In practice, we recommend performing noncollinear calculations in two steps:
</p>
<ul><li>First, calculate the nonmagnetic ground state and generate a <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> and a <a href="/wiki/index.php/CHGCAR" title="CHGCAR">CHGCAR</a> file.</li>
<li>Second, read the <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> and <a href="/wiki/index.php/CHGCAR" title="CHGCAR">CHGCAR</a> file, and supply initial magnetic moments using the <a href="/wiki/index.php/MAGMOM" title="MAGMOM">MAGMOM</a> tag.</li></ul>
<p>We recommend setting <a href="/wiki/index.php/GGA_COMPAT" title="GGA COMPAT">GGA_COMPAT</a> = False and <a href="/wiki/index.php/LASPH" title="LASPH">LASPH</a>= True for noncollinear calculations since this improves the numerical precision of calculations using the generalized-gradient approximation (GGA).
</p>
<p>Consider setting <a href="/wiki/index.php/AMIX_MAG" title="AMIX MAG">AMIX_MAG</a> and <a href="/wiki/index.php/BMIX_MAG" title="BMIX MAG">BMIX_MAG</a> for better convergence when using <a class="mw-redirect" href="/wiki/index.php/Density_mixing" title="Density mixing">density mixing</a>.
</p>
<p>The <a href="/wiki/index.php/I_CONSTRAINED_M" title="I CONSTRAINED M">I_CONSTRAINED_M</a> tag can constrain the on-site magnetic moments.
</p>
<p>Supported as of VASP.4.5.
</p>
