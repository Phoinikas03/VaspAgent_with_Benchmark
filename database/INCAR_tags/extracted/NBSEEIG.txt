<p><a class="mw-selflink selflink">NBSEEIG</a> = [integer]<br/> Default: <b>NBSEEIG</b> = 0 
</p>
<p>Description: <a class="mw-selflink selflink">NBSEEIG</a> sets the number of BSE eigenvectors written to the <a href="/wiki/index.php/BSEFATBAND" title="BSEFATBAND">BSEFATBAND</a> file. If <a href="/wiki/index.php/BSEHOLE" title="BSEHOLE">BSEHOLE</a> or <a href="/wiki/index.php/BSEELECTRON" title="BSEELECTRON">BSEELECTRON</a> is set, <a class="mw-selflink selflink">NBSEEIG</a> exciton wavefunctions are written to the <a href="/wiki/index.php/CHG" title="CHG">CHG</a>.XX files.
</p>
