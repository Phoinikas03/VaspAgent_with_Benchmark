<p>Description: <a class="mw-selflink selflink">SYMPREC</a> determines to which accuracy the positions in the <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> file must be specified (as of VASP.4.4.4).
</p>
<p><a class="mw-selflink selflink">SYMPREC</a> determines how accurately the positions in the <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> file must be specified.
The default, <a class="mw-selflink selflink">SYMPREC</a>=10<sup>-5</sup>, is usually large enough, even if the <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> file has been generated with single precision accuracy.
Increasing <a class="mw-selflink selflink">SYMPREC</a> means that the positions in the <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> file can be specified with less accuracy (increasing fuzziness). Please also have a look at <a href="/wiki/index.php/POSCAR#Precision_and_symmetry" title="POSCAR">this section</a>.
</p>
