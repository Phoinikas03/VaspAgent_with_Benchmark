<p><a class="mw-selflink selflink">AGGAC</a> = [real] 
</p>
<p>Description: <a class="mw-selflink selflink">AGGAC</a> is a parameter that multiplies the gradient correction in the GGA correlation functional.
</p>
<p><a class="mw-selflink selflink">AGGAC</a> can be used as the fraction of gradient correction in the GGA correlation in a Hartree-Fock/DFT hybrid functional. 
</p>
