<p>Description: switches on the <b>k</b>-point projection scheme.
</p>
<p>For <a class="mw-selflink selflink">LKPROJ</a>=.TRUE., VASP will project the orbitals onto the reciprocal space of an alternative unit cell.
This unit cell has to be supplied in the file <a href="/wiki/index.php/POSCAR.prim" title="POSCAR.prim">POSCAR.prim</a>, in the usual <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> format.
</p>
<p>As a first step, the <b>k</b>-projection scheme determines the set {<b>k′</b>}, of <b>k</b>-points in the irreducible part of the first Brillouin zone of the structure given in <a href="/wiki/index.php/POSCAR.prim" title="POSCAR.prim">POSCAR.prim</a>, for which
</p>
<p>where <b>G</b> and <b>G′</b> are reciprocal space vectors in the reciprocal spaces of the structures specified in <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> and <a href="/wiki/index.php/POSCAR.prim" title="POSCAR.prim">POSCAR.prim</a>, respectively. As usual, the set of points {<b>k</b>} is specified in the <a href="/wiki/index.php/KPOINTS" title="KPOINTS">KPOINTS</a> file.
The set {<b>k′</b>} is written to the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a> file. Look at the part of the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a> following <tt>NKPTS_PRIM</tt>. 
</p>
