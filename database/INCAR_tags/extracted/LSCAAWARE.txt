<p><a class="mw-selflink selflink">LSCAAWARE</a> = [logical] 
</p>
<p>Description: <a class="mw-selflink selflink">LSCAAWARE</a> controls the distribution of the Hamilton matrix. 
</p>
<p>For <a class="mw-selflink selflink">LSCAAWARE</a>=.TRUE., VASP distributes the Hamilton matrix among the MPI ranks.  
For <a class="mw-selflink selflink">LSCAAWARE</a>=.FALSE., each MPI ranks allocates the complete Hamiltonain. In both cases <a href="/wiki/index.php/LSCALAPACK" title="LSCALAPACK">LSCALAPACK</a> decides if ScaLAPACK routines are used for diagonalization. 
</p>
