<p><a class="mw-selflink selflink">ML_MCONF_NEW</a> = [integer]<br/> Default: <b>ML_MCONF_NEW</b> = 5 
</p>
<p>Description: This tag sets the number of configurations that are stored temporarily as candidates for the training data in the machine learning force field method.
</p>
<p>The use of this tag in combination with the learning algorithms is described here: <a href="/wiki/index.php/Machine_learning_force_field_calculations:_Basics#Sampling_of_training_data_and_local_reference_configurations" title="Machine learning force field calculations: Basics">here</a>.
If force fields are reparameterized (<code style="padding: 2px"><a href="/wiki/index.php/ML_MODE" title="ML MODE">ML_MODE</a> = select</code>), calculations are usually more efficient if this parameter is increased to values around 10-16 and setting <code style="padding: 2px"><a href="/wiki/index.php/ML_CDOUB" title="ML CDOUB">ML_CDOUB</a> = 4</code>. This is particularly relevant if the ML_AB file is large.
</p>
