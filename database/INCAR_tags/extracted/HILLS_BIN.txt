<p>Description: <a class="mw-selflink selflink">HILLS_BIN</a> sets the number of steps after which the bias potential is updated in a metadynamics run (in case VASP was compiled with <a class="mw-redirect" href="/wiki/index.php/Precompiler_flags" title="Precompiler flags">-Dtbdyn</a>).
</p>
<p>In <a href="/wiki/index.php/Metadynamics" title="Metadynamics">metadynamics</a> (<a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a>=11 | 21), the bias potential is given as 
</p>
<p>Thre parameters (<a href="/wiki/index.php/HILLS_H" title="HILLS H">HILLS_H</a>, <a href="/wiki/index.php/HILLS_W" title="HILLS W">HILLS_W</a>, and <a class="mw-selflink selflink">HILLS_BIN</a>) must be provided by the user.
</p>
<p><br/>
The number of steps after which the bias potential is updated is set by <a class="mw-selflink selflink">HILLS_BIN</a>.
</p>
