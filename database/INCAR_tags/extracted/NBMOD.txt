<p><a class="mw-selflink selflink">NBMOD</a> = -3 | -2 | -1 | 0 | [positive integer] 
</p>
<p>Description: <a class="mw-selflink selflink">NBMOD</a> controls how bands are selected when computing <a class="mw-redirect" href="/wiki/index.php/Band-decomposed_charge_densities" title="Band-decomposed charge densities">partial charge densities</a>. 
</p>
<p><a class="mw-selflink selflink">NBMOD</a> is used with other tags to define the mode of band selection for partial charge densities in <a href="/wiki/index.php/PARCHG" title="PARCHG">PARCHG</a>, <a href="/wiki/index.php/Vaspout.h5" title="Vaspout.h5">vaspout.h5</a>, or <a href="/wiki/index.php/CHGCAR" title="CHGCAR">CHGCAR</a> files. There are several ways to set this tag. 
</p>
<ul><li><a class="mw-selflink selflink">NBMOD</a> = n: Use n bands</li></ul>
<ul><li><a class="mw-selflink selflink">NBMOD</a> = 0: Use all bands</li></ul>
<ul><li><a class="mw-selflink selflink">NBMOD</a> = -1: Use all occupied bands</li></ul>
<ul><li><a class="mw-selflink selflink">NBMOD</a> = -2: Use an absolute energy interval to select contributing bands</li></ul>
