<p>Description: <a class="mw-selflink selflink">SHAKETOL</a> specifies the tolerance for the SHAKE algorithm (in case VASP was compiled with <a href="/wiki/index.php/Precompiler_options#-Dtbdyn" title="Precompiler options">-Dtbdyn</a>).
</p>
<p>Constrained molecular dynamics (<a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a>=1 | 2) are performed using a <a href="/wiki/index.php/MDALGO#SHAKE" title="MDALGO">SHAKE algorithm</a>.<sup class="reference" id="cite_ref-Ryckaert77_1-0"><a href="#cite_note-Ryckaert77-1">[1]</a></sup>
</p>
<p><a class="mw-selflink selflink">SHAKETOL</a> specifies the tolerance for the SHAKE algorithm.
If the error for all geometric constraints does not decrease below this predefined tolerance within the allowed number of iterations (<a href="/wiki/index.php/SHAKEMAXITER" title="SHAKEMAXITER">SHAKEMAXITER</a>), VASP terminates with an error message. 
</p>
