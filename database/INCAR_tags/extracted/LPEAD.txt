<p>When <a class="mw-selflink selflink">LPEAD</a>=.TRUE., VASP will compute |∇<sub><b>k</b></sub>u<sub>n<b>k</b></sub>⟩ using the aforementioned finite difference scheme. The order of the finite difference approximation can be specified by means of the <a href="/wiki/index.php/IPEAD" title="IPEAD">IPEAD</a>-tag (default: <a href="/wiki/index.php/IPEAD" title="IPEAD">IPEAD</a>=4).
</p>
<p>These tags may be used in combination with <a href="/wiki/index.php/LOPTICS" title="LOPTICS">LOPTICS</a>=.TRUE. and <a href="/wiki/index.php/LEPSILON" title="LEPSILON">LEPSILON</a>=.TRUE..
</p>
