<p><a class="mw-selflink selflink">GGA_COMPAT</a> = .TRUE. | .FALSE.<br/> Default: <b>GGA_COMPAT</b> = .TRUE. 
</p>
<p>Description: If set to <a class="mw-selflink selflink">GGA_COMPAT</a> = .<i>FALSE</i>., this tag restores the full lattice symmetry for gradient-corrected functionals.
</p>
<p><a href="/wiki/index.php/GGA" title="GGA">GGA</a> and <a href="/wiki/index.php/METAGGA" title="METAGGA">METAGGA</a> functionals might break the symmetry of
the Bravais lattice slightly for cells that are not primitive cubic cells.
The origin of this problem is subtle and relates to the fact that the gradient field breaks the lattice symmetry for noncubic lattices. This can be fixed by setting
</p>
