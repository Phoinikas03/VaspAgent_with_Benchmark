<p><a class="mw-selflink selflink">CH_LSPEC</a> = [logical]<br/> Default: <b>CH_LSPEC</b> = .FALSE. 
</p>
<p>Description: This flag controls whether the dielecectric function using the supercell core-hole method is calculated or not.
</p>
<p>How to calculate X-ray absorption spectra from the supercell core-hole method are is explained <a href="/wiki/index.php/Supercell_core-hole_calculations" title="Supercell core-hole calculations">here</a>.
</p>
<p>This tag should be used in combination with the following important tags for the core-hole approximation:
</p>
<ul><li><a href="/wiki/index.php/ICORELEVEL" title="ICORELEVEL">ICORELEVEL</a>: To enable core-hole calculations in the final-state approximation with self-consistent field cycles (SCF) one has to set <a href="/wiki/index.php/ICORELEVEL" title="ICORELEVEL">ICORELEVEL</a>=2. Core-hole calculations in the initial-state approximation (<a href="/wiki/index.php/ICORELEVEL" title="ICORELEVEL">ICORELEVEL</a>=1) are also available, but they are physically less relevant and should be only used if especially needed.</li>
<li><a href="/wiki/index.php/CLNT" title="CLNT">CLNT</a>: This tag selects the species holding the core hole. This number corresponds to the species defined in the <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> and <a href="/wiki/index.php/POTCAR" title="POTCAR">POTCAR</a> files.</li>
<li><a href="/wiki/index.php/CLN" title="CLN">CLN</a>: Specifies the <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle n}" xmlns="http://www.w3.org/1998/Math/MathML">

<li><a href="/wiki/index.php/CLL" title="CLL">CLL</a>: Specifies the <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle l}" xmlns="http://www.w3.org/1998/Math/MathML">

<p>And following tags to control the calculation of the dieletric function:
</p>
<ul><li><a href="/wiki/index.php/CH_SIGMA" title="CH SIGMA">CH_SIGMA</a>: The broadening of the spectrum is by default of Gaussian form and the broadening width in eV is set by <a href="/wiki/index.php/CH_SIGMA" title="CH SIGMA">CH_SIGMA</a>. We recommend using a very small broadening <a href="/wiki/index.php/CH_SIGMA" title="CH SIGMA">CH_SIGMA</a><span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle \le}" xmlns="http://www.w3.org/1998/Math/MathML">

Also, the spectrum can be recalculated with different parameters without the need to redo the electronic self-consistent field cycle. For that one can use the converged <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> from the previous calculation and set <a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a>=<i>None</i> together with the new parameters for the spectrum "CH_*" in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file.</li>
<li><a href="/wiki/index.php/CH_NEDOS" title="CH NEDOS">CH_NEDOS</a>: Sets the number of grid points on the energy axis of the spectrum.</li>
<li><a href="/wiki/index.php/CH_AMPLIFICATION" title="CH AMPLIFICATION">CH_AMPLIFICATION</a>: Scaling of the spectrum by the specified value. This tag is not important but can be useful sometimes if one needs to scale the spectrum a priori. Otherwise, it is recommended to scale the spectrum a posteriori.</li></ul>
