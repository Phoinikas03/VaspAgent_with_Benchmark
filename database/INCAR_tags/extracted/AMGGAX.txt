<p><a class="mw-selflink selflink">AMGGAX</a> = [real] 
</p>
<p>Description: <a class="mw-selflink selflink">AMGGAX</a> is a parameter that multiplies the meta-GGA exchange functional (available as of VASP.6.4.0).
</p>
<p><a class="mw-selflink selflink">AMGGAX</a> can be used as the fraction of meta-GGA exchange in a Hartree-Fock/DFT hybrid functional (possible since VASP.6.4.0).
</p>
