<p><a class="mw-selflink selflink">FMP_SWAPNUM</a> = integer<br/> Default: <b>FMP_SWAPNUM</b> =  1 
</p>
<p>Description: Number of pairs that are exchanged in a single swapping event in the <a href="/wiki/index.php/M%C3%BCller-Plathe_method" title="Müller-Plathe method">Müller-Plathe method</a>.
</p>
<p><a class="mw-selflink selflink">FMP_SWAPNUM</a> defines the number of pairs of particles exchanged in a single swapping event of the reverse nonequilibrium molecular dynamics run using the <a href="/wiki/index.php/M%C3%BCller-Plathe_method" title="Müller-Plathe method">Müller-Plathe method</a>. Only the particles of the same type are selected. 
</p>
