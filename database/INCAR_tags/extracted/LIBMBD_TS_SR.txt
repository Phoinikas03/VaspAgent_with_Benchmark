The value is internally passed to the libMBD input <b>ts_sr</b> described at the page <sup class="reference" id="cite_ref-libmbd_input_5-0"><a href="#cite_note-libmbd_input-5">[5]</a></sup>. <a class="mw-selflink selflink">LIBMBD_TS_SR</a> is similar to the <a href="/wiki/index.php/VDW_SR" title="VDW SR">VDW_SR</a> tag that is used for the VASP implementation of the Tkatchenko-Scheffler method.
</p>
<p>libMBD is a separate library package that has to be downloaded<sup class="reference" id="cite_ref-libmbd_2_3-2"><a href="#cite_note-libmbd_2-3">[3]</a></sup> and compiled before VASP is compiled with the corresponding <a href="/wiki/index.php/Precompiler_options#-DLIBMBD" title="Precompiler options">precompiler options</a> and <a href="/wiki/index.php/Makefile.include#LIBMBD_.28optional.29" title="Makefile.include">links to the libraries</a>.
</p>
