<p><a class="mw-selflink selflink">ML_IAFILT2</a> = [integer]<br/> Default: <b>ML_IAFILT2</b> = 2 
</p>
<p>Description: This tag specifies the type of angular filtering used in the machine learning force field method.
</p>
<p>This tag is only used if <a href="/wiki/index.php/ML_LAFILT2" title="ML LAFILT2">ML_LAFILT2</a>=<i>.TRUE.</i> is set.
</p>

<p>In the case of the angular descriptor two radial basis functions are multiplied by each other (see <a class="mw-redirect" href="/wiki/index.php/On-the-fly_machine_learning_force_field_generation_using_Bayesian_linear_regression#Basis_set_expansion" title="On-the-fly machine learning force field generation using Bayesian linear regression">here</a>). Both basis functions use the same filtering function and hence the filtering is done by the square of the filtering function. This is plotted in Fig. 1 for the two different functions used for <a class="mw-selflink selflink">ML_IAFILT2</a>=1 and 2 (labeled as TYPE1 and TYPE2, respectively). In the case of <a class="mw-selflink selflink">ML_IAFILT2</a>=2 it can be seen that for the filtering parameter <a href="/wiki/index.php/ML_AFILT2" title="ML AFILT2">ML_AFILT2</a>=0.002 and <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle l}" xmlns="http://www.w3.org/1998/Math/MathML">
<semantics>
<mrow class="MJX-TeXAtom-ORD">
<mstyle displaystyle="true" scriptlevel="0">
<mi>l</mi>
</mstyle>
</mrow>
<annotation encoding="application/x-tex">{\displaystyle l}</annotation>
</semantics>
</math></span><img alt="{\displaystyle l}" aria-hidden="true" class="mwe-math-fallback-image-inline" src="/wiki/index.php?title=Special:MathShowImage&amp;hash=2db95e8e1a9267b7a1188556b2013b33&amp;mode=mathml" style="vertical-align: -0.338ex; height: 2.176ex; width: 0.693ex;"/></span>=5 the function has only a contribution of 0.15. Using this filtering parameter the maximum cut-off for the angular quantum number can be reduced to <a href="/wiki/index.php/ML_LMAX2" title="ML LMAX2">ML_LMAX2</a>=4.
