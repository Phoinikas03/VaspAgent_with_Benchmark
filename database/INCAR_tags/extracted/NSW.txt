</p>
<p>Description: <a class="mw-selflink selflink">NSW</a> sets the maximum number of ionic steps.
</p>
<p><a href="/wiki/index.php/IBRION" title="IBRION">IBRION</a> = 0:
</p>
<p><a href="/wiki/index.php/IBRION" title="IBRION">IBRION</a> != 0: 
</p>
<p>Within each ionic step at most <a href="/wiki/index.php/NELM" title="NELM">NELM</a> electronic steps are performed. It is fewer if the convergence criterium set by <a href="/wiki/index.php/EDIFF" title="EDIFF">EDIFF</a> is met before. Forces and stresses are calculated according to the setting of <a href="/wiki/index.php/ISIF" title="ISIF">ISIF</a> for each ionic step.
</p>
