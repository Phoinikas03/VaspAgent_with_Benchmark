<a class="mw-selflink selflink">WANNIER90_WIN</a> = [string] 
</p>
<p>Description: <a class="mw-selflink selflink">WANNIER90_WIN</a> sets the content of the <b>wannier90.win</b> file.
</p>
<p>The <a class="mw-selflink selflink">WANNIER90_WIN</a> tag is a multiline string, where the content of the <b>wannier90.win</b> file can be specified. For instance,
</p>
<p>Additionally, the value of some <a class="external text" href="https://github.com/wannier-developers/wannier90/raw/v3.1.0/doc/compiled_docs/user_guide.pdf" rel="nofollow">Wannier90 tags</a>  is set automatically based on the VASP calculation, e.g., <i>kpoints</i>, <i>atoms</i>, <i>unit_cell</i>, <i>mp_grid</i>, <i>spinors</i>, <i>num_bands</i>, <i>num_wann</i>. 
</p>
<p>Available as of VASP 6.2.0.
