<p><a class="mw-selflink selflink">LIBMBD_N_OMEGA_GRID</a> = [integer]<br/> Default: <b>LIBMBD_N_OMEGA_GRID</b> = 15 (default in libMBD) 
</p>
<p>Description: <a class="mw-selflink selflink">LIBMBD_N_OMEGA_GRID</a> sets the number of points on the grid of imaginary frequencies used in the library libMBD of many-body dispersion methods<sup class="reference" id="cite_ref-libmbd_1_1-0"><a href="#cite_note-libmbd_1-1">[1]</a></sup><sup class="reference" id="cite_ref-libmbd_2_2-0"><a href="#cite_note-libmbd_2-2">[2]</a></sup><sup class="reference" id="cite_ref-hermann:jcp:2023_3-0"><a href="#cite_note-hermann:jcp:2023-3">[3]</a></sup>.
</p>
<p><a class="mw-selflink selflink">LIBMBD_N_OMEGA_GRID</a> allows to choose the number of points on the grid of imaginary frequencies used in the library libMBD of many-body dispersion methods. The value is internally passed to the libMBD input <b>n_omega_grid</b> described at the page <sup class="reference" id="cite_ref-libmbd_input_4-0"><a href="#cite_note-libmbd_input-4">[4]</a></sup>.
</p>
<p>libMBD is a separate library package that has to be downloaded<sup class="reference" id="cite_ref-libmbd_2_2-1"><a href="#cite_note-libmbd_2-2">[2]</a></sup> and compiled before VASP is compiled with the corresponding <a href="/wiki/index.php/Precompiler_options#-DLIBMBD" title="Precompiler options">precompiler options</a> and <a href="/wiki/index.php/Makefile.include#LIBMBD_.28optional.29" title="Makefile.include">links to the libraries</a>.
</p>
