<p><a class="mw-selflink selflink">ZVAL</a> = [real]<br/> Default: <b>ZVAL</b> = values read from <a href="/wiki/index.php/POTCAR" title="POTCAR">POTCAR</a> 
</p>
<p>Description: <a class="mw-selflink selflink">ZVAL</a> describes the valency of each atomic sphere.
</p>
<p><a class="mw-selflink selflink">ZVAL</a> determines the valency of each atomic species and should be omitted usually since the values are read from the <a href="/wiki/index.php/POTCAR" title="POTCAR">POTCAR</a> file. If incompatibilities exist, VASP will stop.
</p>
