<p><a class="mw-selflink selflink">NBANDSGW</a> = [integer]<br/> Default: <b>NBANDSGW</b> =  twice the number of occupied states 
</p>
<p>Description: The flag determines how many QP energies are calculated and updated in GW type calculations. 
</p>
<p>This value usually needs to be increased somewhat for partially or fully self-consistent calculations. Very accurate results
are only obtained when <a class="mw-selflink selflink">NBANDSGW</a> approaches <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a>, although this  dramatically increases the computational requirements.
</p>
