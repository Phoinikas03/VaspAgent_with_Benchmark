<p><a class="mw-selflink selflink">LWAVEH5</a> = [logical]<br/> Default: <b>LWAVEH5</b> = <a href="/wiki/index.php/LH5" title="LH5">LH5</a> 
</p>
<p>Description: Determines whether the wavefunctions are written to <a href="/wiki/index.php/Vaspwave.h5" title="Vaspwave.h5">vaspwave.h5</a> file at the end of a run.
</p>
<p>If <code style="padding: 2px"><a class="mw-selflink selflink">LWAVEH5</a> = True</code>, the Kohn-Sham orbitals, i.e., wavefunctions, are written to the <a href="/wiki/index.php/Vaspwave.h5" title="Vaspwave.h5">vaspwave.h5</a> file at the end of the calculation. This can be used to restart VASP.
</p>
