<p>The <a href="/wiki/index.php/POTCAR" title="POTCAR">POTCAR</a> files contain a default <a class="mw-redirect" href="/wiki/index.php/ENMAX" title="ENMAX">ENMAX</a> (and <a href="/wiki/index.php/ENMIN" title="ENMIN">ENMIN</a>). Therefore, it is, in principle, not necessary to specify <a class="mw-selflink selflink">ENCUT</a> in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file.
For calculations with more than one species, the maximum cutoff <a class="mw-redirect" href="/wiki/index.php/ENMAX" title="ENMAX">ENMAX</a> (or <a href="/wiki/index.php/ENMIN" title="ENMIN">ENMIN</a>) value is used for the calculation (see <a href="/wiki/index.php/PREC" title="PREC">PREC</a>).
