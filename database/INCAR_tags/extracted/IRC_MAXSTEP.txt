<p><a class="mw-selflink selflink">IRC_MAXSTEP</a> = real<br/> Default: <b>IRC_MAXSTEP</b> =  3.000 
</p>
<p>Description: Upper limit for the step size (in fs).
</p>
<p>The damped-velocity-Verlet algorithm for the <a class="mw-redirect" href="/wiki/index.php/IRC_calculations" title="IRC calculations">IRC calculations</a> uses an adaptively varying size of the time step. It depends on the estimated accuracy of the previous step. <a class="mw-selflink selflink">IRC_MAXSTEP</a> defines the upper limit for the step size in fs.
</p>
