<p><a class="mw-selflink selflink">ML_EATOM_REF</a> = [real array]<br/> Default: <b>ML_EATOM_REF</b> = 0.0 
</p>
<p>Description: Reference total energies of isolated atoms used in the machine learning force field method.
</p>
<p>This tag is only used if <code style="padding: 2px"><a href="/wiki/index.php/ML_ISCALE_TOTEN" title="ML ISCALE TOTEN">ML_ISCALE_TOTEN</a> = 1</code>.
</p>
<p>If <a class="mw-selflink selflink">ML_EATOM_REF</a> is not provided in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file then 0.0 is assumed for all species in the system.
</p>
<p>By default this tag is not used since all energies are scaled to the average of the training data (<code style="padding: 2px"><a href="/wiki/index.php/ML_ISCALE_TOTEN" title="ML ISCALE TOTEN">ML_ISCALE_TOTEN</a> = 2</code>).
</p>
<p>If this tag is used, each reference energy should be obtained from a VASP calculation of an isolated atoms in a sufficiently large simulation box. The reference is then simply taken from the Helmholtz free energy from the <a href="/wiki/index.php/OSZICAR" title="OSZICAR">OSZICAR</a> file (value following "1 F=" in that file). 
</p>

</p>
<p>The unit of the energies is eV/atom. 
</p>
