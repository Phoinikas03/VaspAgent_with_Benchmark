<p><a class="mw-selflink selflink">APACO</a> = [real]<br/> Default: <b>APACO</b> = 10 
</p>
<p>Description: <a class="mw-selflink selflink">APACO</a> sets the maximum distance in the evaluation of the pair-correlation function (in Å).
</p>
<p>VASP evaluates the pair-correlation function each <a href="/wiki/index.php/NBLOCK" title="NBLOCK">NBLOCK</a> steps and writes it after <a href="/wiki/index.php/NBLOCK" title="NBLOCK">NBLOCK</a>×<a href="/wiki/index.php/KBLOCK" title="KBLOCK">KBLOCK</a> steps to the <a href="/wiki/index.php/PCDAT" title="PCDAT">PCDAT</a> file.
</p>
