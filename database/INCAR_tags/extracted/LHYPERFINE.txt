<h2><span class="mw-headline" id="Advice">Advice</span></h2>
<h3><span class="mw-headline" id="PAW_pseudopotentials">PAW pseudopotentials</span></h3>
<ul><li><a href="/wiki/index.php/Choosing_pseudopotentials" title="Choosing pseudopotentials">Choice of PAW potentials</a>: The hyperfine coupling parameter can be sensitive to the specific <a href="/wiki/index.php/Category:Pseudopotentials" title="Category:Pseudopotentials">PAW potential</a> used, as different <a href="/wiki/index.php/Category:Pseudopotentials" title="Category:Pseudopotentials">pseudopotentials</a> include a varying number of electrons in the valence (i.e. _sv, _pv). It is important to match the all-electron (AE) wavefunction.</li>
<li><a href="/wiki/index.php/Available_pseudopotentials" title="Available pseudopotentials">GW pseudopotentials</a> (i.e. _GW) often offer a better description than standard potentials.</li></ul>
<h3><span class="mw-headline" id="Core_contributions_and_hybrid_functionals">Core contributions and hybrid functionals</span></h3>
<ul><li>Make sure to include core contributions when reporting Fermi contact terms <sup class="reference" id="cite_ref-yazyev:prb:2005_3-1"><a href="#cite_note-yazyev:prb:2005-3">[3]</a></sup>. These are very important and their inclusion improves comparison to experiment <sup class="reference" id="cite_ref-szasz:prb:2013_1-2"><a href="#cite_note-szasz:prb:2013-1">[1]</a></sup>.</li>
<li>The use of hybrid functionals can also improve the hyperfine coupling constants when compared to experiment. E.g., for defects in silicon, HSE06 localizes the defect relative to PBE, significantly improving the description <sup class="reference" id="cite_ref-szasz:prb:2013_1-3"><a href="#cite_note-szasz:prb:2013-1">[1]</a></sup>.</li></ul>
<h3><span class="mw-headline" id="Input_parameters">Input parameters</span></h3>
<ul><li>We recommend using tightly converged settings:</li></ul>
<ul><li>Additionally, we recommend performing convergence tests with respect to the plane-wave energy cutoff <a href="/wiki/index.php/ENCUT" title="ENCUT">ENCUT</a> and k-point mesh <a href="/wiki/index.php/KPOINTS" title="KPOINTS">KPOINTS</a> to ensure convergence has been achieved for your system.
<ul><li>Convergence is generally quick with respect to energy cutoff.</li>
<li>Convergence can be very dependent on k-point mesh density. In some cases, e.g. if your cell is too small, then increasing the k-point mesh can result in convergence to a non-magnetic solution, eliminating the hyperfine coupling.</li></ul></li>
<li>Test your system with <a href="/wiki/index.php/LASPH" title="LASPH">LASPH</a> = .TRUE. and .FALSE. In some cases, non-spherical contributions may be important.</li></ul>
<h3><span class="mw-headline" id="Converging_to_non-magnetic_states">Converging to non-magnetic states</span></h3>
<ul><li>It is possible that your system relaxes to a non-magnetic solution, causing the hyperfine splitting to disappear (i.e. all zeros). If you think your system should be magnetic, you can enforce it using <a href="/wiki/index.php/NUPDOWN" title="NUPDOWN">NUPDOWN</a>, which will return the hyperfine splitting, cf. forum post: <a class="external free" href="https://vasp.at/forum/viewtopic.php?t=16921" rel="nofollow">https://vasp.at/forum/viewtopic.php?t=16921</a>. <a href="/wiki/index.php/NUPDOWN" title="NUPDOWN">NUPDOWN</a> will change the <code>Total magnetic moment S=</code> at the start of the hyperfine coupling section in the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a>.</li></ul>
