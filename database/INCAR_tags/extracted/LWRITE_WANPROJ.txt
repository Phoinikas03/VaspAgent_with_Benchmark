<p><a class="mw-selflink selflink">LWRITE_WANPROJ</a> = [logical]<br/> Default: <b>LWRITE_WANPROJ</b> = .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LWRITE_WANPROJ</a> determines whether the Wannier projection file <a href="/wiki/index.php/WANPROJ" title="WANPROJ">WANPROJ</a> is written.
</p>
<p>If <a href="/wiki/index.php/WANPROJ" title="WANPROJ">WANPROJ</a> is present, VASP skips the Wannier projection and reads the effective projectors U<sub>nk</sub> from the <a href="/wiki/index.php/WANPROJ" title="WANPROJ">WANPROJ</a> file for each k point. 
</p>
