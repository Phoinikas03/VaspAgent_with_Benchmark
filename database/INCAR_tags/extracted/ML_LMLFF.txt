<p><a class="mw-selflink selflink">ML_LMLFF</a> = [logical]<br/> Default: <b>ML_LMLFF</b> = .FALSE. 
</p>
<p>Description: Main control tag which enables/disables the use of machine learning force fields.
</p>
<p>If <a class="mw-selflink selflink">ML_LMLFF</a> = .FALSE. machine learning force fields are disabled and all related <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> tags, i.e. all tags starting with "<b>ML_</b>", are ignored. If machine learning force fields are used by setting <a class="mw-selflink selflink">ML_LMLFF</a> = .TRUE., the VASP mode of operation depends on the choice of <a href="/wiki/index.php/ML_MODE" title="ML MODE">ML_MODE</a>. If <a href="/wiki/index.php/ML_MODE" title="ML MODE">ML_MODE</a> is not supplied in the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file then the default mode of operation is to run an MD simulation with on-the-fly machine learning, i.e., <a href="/wiki/index.php/ML_MODE" title="ML MODE">ML_MODE</a>. This training is started "from scratch" if no <a href="/wiki/index.php/ML_AB" title="ML AB">ML_AB</a> file is provided, otherwise a continuation run is performed.
</p>
