<p><a class="mw-selflink selflink">XC_C</a> = [real array]<br/> Default: <b>XC_C</b> =  1.0*NXC 
</p>
<p>Description: Multiplication factors for the components of the functional given by the <a href="/wiki/index.php/XC" title="XC">XC</a> tag.
</p>
<p><a class="mw-selflink selflink">XC_C</a> sets the factors that multiply each component of the functional specified with the <a href="/wiki/index.php/XC" title="XC">XC</a> tag. The number of values specified with <a class="mw-selflink selflink">XC_C</a> has to be equal to the number of functional components set with <a href="/wiki/index.php/XC" title="XC">XC</a> (NXC). Examples of how to use <a class="mw-selflink selflink">XC_C</a> are provided at <a href="/wiki/index.php/XC" title="XC">XC</a>.
</p>
