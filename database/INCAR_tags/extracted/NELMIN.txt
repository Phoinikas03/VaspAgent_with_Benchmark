<p><a class="mw-selflink selflink">NELMIN</a> = [integer]<br/> Default: <b>NELMIN</b> = 2 
</p>
<p>Description: <a class="mw-selflink selflink">NELMIN</a> specifies the minimum number of electronic self-consistency steps. 
</p>
<p>It is rarely necessary to change the value of the <a class="mw-selflink selflink">NELMIN</a> tag. In some cases, e.g., in <a href="/wiki/index.php/Category:Molecular_dynamics" title="Category:Molecular dynamics">molecular-dynamics</a> runs, or <a href="/wiki/index.php/Category:Ionic_minimization" title="Category:Ionic minimization">ionic-minimization</a> methods, we recommend increasing the <a class="mw-selflink selflink">NELMIN</a> tag to a value between 4 and 8.
</p>
