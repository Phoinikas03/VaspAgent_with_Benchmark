<p><a class="mw-selflink selflink">LBLUEOUT</a> = .TRUE.|.FALSE.<br/> Default: <b>LBLUEOUT</b> = .FALSE. 
</p>
<p>Description: for <a class="mw-selflink selflink">LBLUEOUT</a>=.TRUE., VASP writes output for the free-energy gradient calculation to the <a href="/wiki/index.php/REPORT" title="REPORT">REPORT</a> file (in case VASP was compiled with <a class="mw-redirect" href="/wiki/index.php/Precompiler_flags" title="Precompiler flags">-Dtbdyn</a>).
</p>
<p>If LBLUEOUT=.TRUE., the information needed to compute the free-energy gradient is written in the <a href="/wiki/index.php/REPORT" title="REPORT">REPORT</a> file after each molecular-dynamics step (<a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a>=1 | 2), check the lines after the header:
</p>
<p>For the theory of the blue-moon ensemble we refer to <a class="mw-redirect" href="/wiki/index.php/Blue-moon_ensemble" title="Blue-moon ensemble">here</a>.
</p>
