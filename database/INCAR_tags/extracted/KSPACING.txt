<p><a class="mw-selflink selflink">KSPACING</a> = [real]<br/> Default: <b>KSPACING</b> = 0.5 
</p>
<p>Description: The tag  <a class="mw-selflink selflink">KSPACING</a> determines the number of k points if the <a href="/wiki/index.php/KPOINTS" title="KPOINTS">KPOINTS</a> file
is not present.
</p>
