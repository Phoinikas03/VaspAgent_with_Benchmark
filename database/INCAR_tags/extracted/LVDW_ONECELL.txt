<p><a class="mw-selflink selflink">LVDW_ONECELL</a> = [logical][logical][logical]<br/> Default: <b>LVDW_ONECELL</b> =  .FALSE. .FALSE. .FALSE.  
</p>
<p>Description: <a class="mw-selflink selflink">LVDW_ONECELL</a>  can be used to disable vdW interaction with mirror image in X Y Z direction. This is advisable for molecular calculations in the gas phase. In all other cases, use the default.
</p>
<p>Note: There is some confusing documentation on the ASE pages, which states that ".TRUE. .TRUE. .TRUE." enables the interaction with neighboring cells. However, the opposite is the case and <i>.TRUE.</i> disables the interaction (".FALSE. .FALSE. .FALSE." = interactions switched on, ".TRUE. .TRUE. .TRUE." = interactions switched off).
</p>
