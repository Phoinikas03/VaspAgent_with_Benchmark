<p>Description: Number of grid points for the electronic <a class="mw-redirect" href="/wiki/index.php/Density_of_states" title="Density of states">density of states</a> (DOS) and <a class="mw-redirect" href="/wiki/index.php/Dielectric_function" title="Dielectric function">dielectric function</a>.
</p>
<p>The energy range between <a href="/wiki/index.php/EMIN" title="EMIN">EMIN</a> and <a href="/wiki/index.php/EMAX" title="EMAX">EMAX</a> is divided into
<a class="mw-selflink selflink">NEDOS</a> intervals to obtain the grid points. The DOS for the corresponding energy is written in the <a href="/wiki/index.php/DOSCAR" title="DOSCAR">DOSCAR</a> file.
</p>
<p>The smallest peak widths from the dispersion of the respective bands can be estimated by having a look at the Kohn-Sham eigenvalues written in <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a>. <a class="mw-selflink selflink">NEDOS</a> has to be chosen sufficiently large to resolve this dispersion. In addition, the
energy interval defined by <a href="/wiki/index.php/EMIN" title="EMIN">EMIN</a> and <a href="/wiki/index.php/EMAX" title="EMAX">EMAX</a> can be modified.
</p>
<p><a class="mw-selflink selflink">NEDOS</a> is also used to set the total number of frequency points when calculating the <a class="mw-redirect" href="/wiki/index.php/Dielectric_function" title="Dielectric function">dielectric function</a>. 
</p>
