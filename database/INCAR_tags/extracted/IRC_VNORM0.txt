<p><a class="mw-selflink selflink">IRC_VNORM0</a> = real<br/> Default: <b>IRC_VNORM0</b> =  0.002 
</p>
<p>Description: The constant velocity vector (in Å/fs) in an <a class="mw-redirect" href="/wiki/index.php/IRC_calculations" title="IRC calculations">IRC calculation</a>.
</p>
<p>In the damped-velocity-Verlet algorithm for <a class="mw-redirect" href="/wiki/index.php/IRC_calculations" title="IRC calculations">IRC calculations</a>, the damping is realized via rescaling the velocity vector to a constant value (in Å/fs) defined via the parameter <a class="mw-selflink selflink">IRC_VNORM0</a>.
</p>
