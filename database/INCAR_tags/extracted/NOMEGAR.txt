<p><a class="mw-selflink selflink">NOMEGAR</a> = [integer] 
</p>
<p>Description: <a class="mw-selflink selflink">NOMEGAR</a> specifies the number of frequency grid points along the real axis.
</p>
<p>Usually <a class="mw-selflink selflink">NOMEGAR</a> equals <a href="/wiki/index.php/NOMEGA" title="NOMEGA">NOMEGA</a>. If <a class="mw-selflink selflink">NOMEGAR</a> is smaller than  <a href="/wiki/index.php/NOMEGA" title="NOMEGA">NOMEGA</a> (for instance 0), frequencies along the imaginary time axis are included (this feature is currently not fully supported). 
</p>
