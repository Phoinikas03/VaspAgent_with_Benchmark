<p><a class="mw-selflink selflink">NBANDSEXACT</a> = [integer] 
</p>
<p>Description: <a class="mw-selflink selflink">NBANDSEXACT</a> specifies the number of bands used in the all-in-one mode of <a class="mw-redirect" href="/wiki/index.php/Many-body_perturbation_theory" title="Many-body perturbation theory">many-body perturbation theory</a> calculations.  
</p>
<p>In the all-in-one mode, VASP automatically performs the necessary DFT steps prior to the many-body perturbation theory (MBPT) calculation, i.e. a DFT calculation with <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a>, followed by an exact diagonalization of the Kohn-Sham Hamiltonian with <a class="mw-selflink selflink">NBANDSEXACT</a> bands. 
Note, <a class="mw-selflink selflink">NBANDSEXACT</a> is set by default to the maximum number of plane-waves given by the chosen energy cutoff for the orbitals <a href="/wiki/index.php/ENCUT" title="ENCUT">ENCUT</a>. 
In the all-in-one mode, the actual GW/RPA calculation is also performed with <a class="mw-selflink selflink">NBANDSEXACT</a> bands. 
If <a href="/wiki/index.php/NBANDS_WAVE" title="NBANDS WAVE">NBANDS_WAVE</a> is not set, all orbitals are written to <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a>, which potentially becomes huge in file size.
</p>
<p>The all-in-one mode is automatically enabled for <a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a>=[EV]GW[0]R, RPA[R],ACFDT[R] if <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a> is not set.
</p>
