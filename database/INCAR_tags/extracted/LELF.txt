<p><a class="mw-selflink selflink">LELF</a> = .TRUE. | .FALSE.<br/> Default: <b>LELF</b> = .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LELF</a> determines whether to create an
<a href="/wiki/index.php/ELFCAR" title="ELFCAR">ELFCAR</a> file or not.
</p>
<p>This file contains the so-called ELF (electron localization
function) <sup class="reference" id="cite_ref-silvi:nature:371_1-0"><a href="#cite_note-silvi:nature:371-1">[1]</a></sup>
</p>
<p>If <a class="mw-selflink selflink">LELF</a> is set, <a href="/wiki/index.php/NPAR" title="NPAR">NPAR</a>=1 has to be set explicitely in
the <a href="/wiki/index.php/INCAR" title="INCAR">INCAR</a> file in addition
</p>
