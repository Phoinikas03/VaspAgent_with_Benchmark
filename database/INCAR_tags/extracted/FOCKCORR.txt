</p>
<p>Description: The tag <a class="mw-selflink selflink">FOCKCORR</a> determines how the Coulomb convergence corrections are applied.
</p>
<p>The Coulomb potential in reciprocal space 
</p>
<p>diverges for small G vectors.
To alleviate this issue and improve the convergence of the exact exchange integral with respect to supercell size (or k-point mesh density) different methods have been proposed: the auxiliary function methods<sup class="reference" id="cite_ref-gygi:prb:86_1-0"><a href="#cite_note-gygi:prb:86-1">[1]</a></sup>, probe-charge Ewald <sup class="reference" id="cite_ref-massidda:prb:93_2-0"><a href="#cite_note-massidda:prb:93-2">[2]</a></sup> (<a href="/wiki/index.php/HFALPHA" title="HFALPHA">HFALPHA</a>), and Coulomb truncation methods<sup class="reference" id="cite_ref-spenceralavi:prb:08_3-0"><a href="#cite_note-spenceralavi:prb:08-3">[3]</a></sup> (<a href="/wiki/index.php/HFRCUT" title="HFRCUT">HFRCUT</a>).
These mostly involve modifying the Coulomb Kernel in a way that yields the same result as the unmodified kernel within the limit of large supercell sizes.
</p>

</p>
<p>with <span class="mwe-math-element"><span class="mwe-math-mathml-inline mwe-math-mathml-a11y" style="display: none;"><math alttext="{\displaystyle \chi}" xmlns="http://www.w3.org/1998/Math/MathML">

or by including the original orbital scaled by the convergence correction when <a class="mw-selflink selflink">FOCKCORR</a>=2
</p>
<p>For Hartree-Fock or hybrid functional calculations, either <a class="mw-selflink selflink">FOCKCORR</a>=1 or <a class="mw-selflink selflink">FOCKCORR</a>=2 can be used and should yield the same results when <a class="mw-redirect" href="/wiki/index.php/LMAXFOCKAE" title="LMAXFOCKAE">LMAXFOCKAE</a>=-1 and there are no aliasing errors in the exact exchange (see <a href="/wiki/index.php/PRECFOCK" title="PRECFOCK">PRECFOCK</a> for more details).
For post-DFT methods such as ACDFT, GW, and BSE the <a class="mw-selflink selflink">FOCKCORR</a>=2 should be used because the overlap densities are reconstructed in the plane-wave grid (see <a class="mw-redirect" href="/wiki/index.php/LMAXFOCKAE" title="LMAXFOCKAE">LMAXFOCKAE</a> tag).
</p>
