<p><a class="mw-selflink selflink">LANGEVIN_GAMMA</a> = [Real array]<br/> Default: <b>LANGEVIN_GAMMA</b> = NTYP×0 
</p>
<p>Description: <a class="mw-selflink selflink">LANGEVIN_GAMMA</a> specifies the friction coefficients (in ps<sup>-1</sup>) for atomic degrees-of-freedom when using a Langevin thermostat (in case VASP was compiled with <a class="mw-redirect" href="/wiki/index.php/Precompiler_flags" title="Precompiler flags">-Dtbdyn</a>).
</p>
<p>When using a <a href="/wiki/index.php/Langevin_thermostat" title="Langevin thermostat">Langevin thermostat</a><sup class="reference" id="cite_ref-Allen91_1-0"><a href="#cite_note-Allen91-1">[1]</a></sup> (<a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a>=3), the friction coefficients γ for the atomic degrees-of-freedom are specified (in ps<sup>-1</sup>) using the <a class="mw-selflink selflink">LANGEVIN_GAMMA</a>-tag.
</p>
<p>One has to specify a separate friction coefficient for each of the NTYP atomic species found on the <a href="/wiki/index.php/POTCAR" title="POTCAR">POTCAR</a>-file. 
</p>
