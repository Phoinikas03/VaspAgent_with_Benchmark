<p><a class="mw-selflink selflink">LSMP2LT</a> = .FALSE. | .TRUE. 
</p>
<p>Description: <a class="mw-selflink selflink">LSMP2LT</a> selects a stochastic Laplace transformed MP2 algorithm.
</p>
<p>If <a class="mw-selflink selflink">LSMP2LT</a>=.TRUE. and <a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a>=ACFTDRK is set, a quartic scaling stochastic Laplace transformed MP2 algorithm is selected.<sup class="reference" id="cite_ref-schaefer:JCP2017_1-0"><a href="#cite_note-schaefer:JCP2017-1">[1]</a></sup>
</p>
<p>This tag should be used in combination with <a href="/wiki/index.php/KPAR" title="KPAR">KPAR</a> to tweak parallelization as described in this <a href="/wiki/index.php/Stochastic_LTMP2#Parallelization" title="Stochastic LTMP2">tutorial</a>.
</p>
