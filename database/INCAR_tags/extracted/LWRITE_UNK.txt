<p><a class="mw-selflink selflink">LWRITE_UNK</a> = .TRUE. | .FALSE.<br/> Default: <b>LWRITE_UNK</b> = .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LWRITE_UNK</a> decides whether the cell-periodic part of the relevant Bloch functions is written.
</p>
<p>For <a class="mw-selflink selflink">LWRITE_UNK</a>=True, VASP writes the cell-periodic part of the Kohn–Sham orbitals in spin channel s at k point p to the file <b>wannier90.UNKp.s</b>. This file can be used to plot Wannier orbitals with WANNIER90.
</p>
<p>For details on the execution of <tt>wannier_setup</tt> in VASP, see the description of the <a href="/wiki/index.php/LWANNIER90" title="LWANNIER90">LWANNIER90</a> tag.
For information on the <b>wannier90.win</b> file and the execution of WANNIER90, we refer to the <a class="external text" href="http://www.wannier.org/doc/user_guide.pdf" rel="nofollow">WANNIER90 manual</a>.
</p>
