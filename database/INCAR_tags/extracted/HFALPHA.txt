<p><a class="mw-selflink selflink">HFALPHA</a> = [real] 
</p>
<p>Description: <a class="mw-selflink selflink">HFALPHA</a> sets the decay constant used in the method of Massida, Posternak, and Baldereschi, which is activated by <a href="/wiki/index.php/HFRCUT" title="HFRCUT">HFRCUT</a>=0.
</p>
<p><a class="mw-selflink selflink">HFALPHA</a> sets the decay constant in the error-function-like charge distribution for the method of Massida, Posternak, and Baldereschi<sup class="reference" id="cite_ref-massidda:prb:93_1-0"><a href="#cite_note-massidda:prb:93-1">[1]</a></sup>. The error-function-like charge distribution is used to calculate the difference between the isolated probe charge and the periodically repeated probe charge in a homogenous background. The default for <a class="mw-selflink selflink">HFALPHA</a> is 6/sqrt(<a class="mw-redirect" href="/wiki/index.php/ENMAX" title="ENMAX">ENMAX</a>)/(2π) in atomic units. This usually yields robust and accurate results in the range of meV compared to the Ewald summation used for a regular k-mesh. This is the default approach used to implement the convergence corrections of the <a href="/wiki/index.php/Coulomb_singularity" title="Coulomb singularity">Coulomb singularity</a> in Hartree-Fock calculations. This does not work correctly for bandstructure calculations using the <a href="/wiki/index.php/Si_HSE_bandstructure" title="Si HSE bandstructure">0-weight scheme</a> or <a href="/wiki/index.php/KPOINTS_OPT" title="KPOINTS OPT">KPOINTS_OPT</a> because the correction is only applied for points in the regular grid. To overcome this problem we recommend using the Coulomb truncation methods using <a href="/wiki/index.php/HFRCUT" title="HFRCUT">HFRCUT</a>.
</p>
