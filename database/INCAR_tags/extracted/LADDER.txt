<p><a class="mw-selflink selflink">LADDER</a> = [logical]<br/> Default: <b>LADDER</b> =  .NOT. <a href="/wiki/index.php/LRPA" title="LRPA">LRPA</a> 
</p>
<p>Description: Controls whether the ladder diagrams are included in the <a class="mw-redirect" href="/wiki/index.php/BSE" title="BSE">BSE</a> calculation. Note that the default for <a href="/wiki/index.php/LRPA" title="LRPA">LRPA</a> and therefore LADDER is somewhat convoluted; so better to always double-check the <a href="/wiki/index.php/OUTCAR" title="OUTCAR">OUTCAR</a> file whether VASP behaves as expected. Generally, VASP will select ladder diagrams whenever this seems reasonable. This is for instance the case for <a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a>="BSE" or "TDHF" calculations.
</p>
