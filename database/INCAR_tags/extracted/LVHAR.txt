<p>The local potential is written to the <a href="/wiki/index.php/LOCPOT" title="LOCPOT">LOCPOT</a> file and hence to the same file as the local potential for <a href="/wiki/index.php/LVTOT" title="LVTOT">LVTOT</a>=T. Carefully check that the <a href="/wiki/index.php/LOCPOT" title="LOCPOT">LOCPOT</a> file contains the potential you expect.
<a href="/wiki/index.php/WRT_POTENTIAL" title="WRT POTENTIAL">WRT_POTENTIAL</a> also gives access to the ionic and Hartree potentials and offers more options.
