<p><a class="mw-selflink selflink">LIBXC2</a> = [string] or [integer] 
</p>
<p>Description: <a class="mw-selflink selflink">LIBXC2</a> specifies the correlation functional from the library of exchange-correlation functionals Libxc<sup class="reference" id="cite_ref-marques:cpc:2012_1-0"><a href="#cite_note-marques:cpc:2012-1">[1]</a></sup><sup class="reference" id="cite_ref-lehtola:sx:2018_2-0"><a href="#cite_note-lehtola:sx:2018-2">[2]</a></sup><sup class="reference" id="cite_ref-libxc_3-0"><a href="#cite_note-libxc-3">[3]</a></sup>. <a class="mw-selflink selflink">LIBXC2</a> can be used only if the functional specified with <a href="/wiki/index.php/LIBXC1" title="LIBXC1">LIBXC1</a> corresponds to only exchange and not to exchange and correlation.
</p>
<p>A detailed explanation of how to use <a href="/wiki/index.php/LIBXC1" title="LIBXC1">LIBXC1</a> and <a class="mw-selflink selflink">LIBXC2</a> and examples can be found on the <a href="/wiki/index.php/LIBXC1" title="LIBXC1">LIBXC1</a> page.
</p>
