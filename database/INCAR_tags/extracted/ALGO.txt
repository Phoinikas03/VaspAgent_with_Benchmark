<p><a class="mw-selflink selflink">ALGO</a> = Normal | VeryFast | Fast | Conjugate | All | Damped | Subrot | Eigenval | Exact | None | Nothing | CHI | G0W0 | GW0 | GW | scGW0 | scGW | G0W0R | GW0R | GWR | scGW0R | scGWR | ACFDT | RPA | ACFDTR | RPAR | BSE | TDHF<br/> Default: <b>ALGO</b> =  Normal 
</p>
<p>Description: the <a class="mw-selflink selflink">ALGO</a> tag is a convenient option to specify the electronic minimization algorithm (as of VASP.4.5) and/or to select the type of <a class="mw-redirect" href="/wiki/index.php/GW_calculations" title="GW calculations">GW calculations</a>.  
</p>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Normal selects <a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=38 (<a href="/wiki/index.php/Blocked-Davidson_algorithm" title="Blocked-Davidson algorithm">blocked-Davidson-iteration scheme</a>).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=VeryFast selects <a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=48 (<a href="/wiki/index.php/RMM-DIIS" title="RMM-DIIS">RMM-DIIS</a>). This algorithm has been updated for vasp.6 to increase robustness. To select the version that was available in vasp.5, select "Old VeryFast". <a href="/wiki/index.php/Category:Hybrid_functionals" title="Category:Hybrid functionals">hybrid functionals</a> are not supported for <a class="mw-selflink selflink">ALGO</a>=VeryFast. The algorithm can be combined with <a href="/wiki/index.php/LDIAG" title="LDIAG">LDIAG</a>=.FALSE. to conserve the initial orbital order (when orbitals are read from the <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> file).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Old VeryFast (or "ov" or "vo") selects <a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=48 (<a href="/wiki/index.php/RMM-DIIS" title="RMM-DIIS">RMM-DIIS</a>). This option is available in vasp.6 and selects the version of the <a href="/wiki/index.php/RMM-DIIS" title="RMM-DIIS">RMM-DIIS</a> algorithm that was available in vasp.5.</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Fast selects a fairly robust mixture of the <a href="/wiki/index.php/Blocked-Davidson_algorithm" title="Blocked-Davidson algorithm">blocked-Davidson</a> and <a href="/wiki/index.php/RMM-DIIS" title="RMM-DIIS">RMM-DIIS</a> algorithms. In this case, <a href="/wiki/index.php/Blocked-Davidson_algorithm" title="Blocked-Davidson algorithm">blocked Davidson</a> (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=38) is used for the initial phase, and then VASP switches to <a href="/wiki/index.php/RMM-DIIS" title="RMM-DIIS">RMM-DIIS</a> (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=48). Subsequently, for each ionic update, one <a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=38 sweep is performed for each ionic step (except the first one).  This algorithm has been updated for vasp.6 to increase robustness. To select the version that was available in vasp.5, select "Old Fast".</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Old Fast (or "of" or "fo"). This option is available in vasp.6 and selects the <a class="mw-selflink selflink">ALGO</a>=Fast algorithm that was available in vasp.5.</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Conjugate or <a class="mw-selflink selflink">ALGO</a>=All selects an "all band simultaneous update of orbitals" (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=58, both tags are synonymous).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Damped selects a damped velocity friction algorithm (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=53). The algorithm can be combined with <a href="/wiki/index.php/LDIAG" title="LDIAG">LDIAG</a>=.FALSE. to conserve the initial orbital order (when orbitals are read from the <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> file).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Exact performs an exact diagonalization (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=90).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Subrot selects subspace rotation or diagonalization in the sub-space spanned by the orbitals (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=4).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Eigenval allows to recalculate one-electron energies, the density of state and perform selected postprocessing using the current orbitals (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=3) e.g. read from the <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> file.</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=None or <a class="mw-selflink selflink">ALGO</a>=Nothing allows to recalculate the density of states or perform selected postprocessing, using the current orbitals and one electron energies (<a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>=2) e.g. read from the <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> file.</li></ul>
<p>More details can be found under the documentation for the tag <a href="/wiki/index.php/IALGO" title="IALGO">IALGO</a>.
</p>
<p>Conjugate, Subrot, Eigenval, None and Nothing are only supported by vasp.5.2.9 and newer versions.
</p>

<h2><span id="ALGO_for_response_functions_and_GW_calculations_and_ACFDT.2FRPA_calculations"></span><span class="mw-headline" id="ALGO_for_response_functions_and_GW_calculations_and_ACFDT/RPA_calculations">ALGO for response functions and <a class="mw-redirect" href="/wiki/index.php/GW_calculations" title="GW calculations">GW calculations</a> and <a href="/wiki/index.php/ACFDT/RPA_calculations" title="ACFDT/RPA calculations">ACFDT/RPA calculations</a></span></h2>
<p>The following tags are available as of VASP.5.X. 
</p>
<ul><li><a class="mw-selflink selflink">ALGO</a>=CHI calculates the response functions only.</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=TDHF selects TDHF calculations using the VASP internal Cassida code see <a class="mw-redirect" href="/wiki/index.php/BSE_calculations" title="BSE calculations">BSE calculations</a>, (available as of VASP.5.2.12)</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=BSE selects BSE calculations using the VASP internal Cassida code see <a class="mw-redirect" href="/wiki/index.php/BSE_calculations" title="BSE calculations">BSE calculations</a>, (available as of VASP.5.4.1)</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=Timeev performs a delta-pulse in time and then performs <a class="mw-redirect" href="/wiki/index.php/Timepropagation" title="Timepropagation">timepropagation</a></li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=ACFDT selects RPA total energy calculations see <a href="/wiki/index.php/ACFDT/RPA_calculations" title="ACFDT/RPA calculations">ACFDT/RPA calculations</a></li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=RPA synonymous to ACFDT see <a href="/wiki/index.php/ACFDT/RPA_calculations" title="ACFDT/RPA calculations">ACFDT/RPA calculations</a> (available as of VASP.5.3.1)</li></ul>
<p>GW tags have been renamed in VASP as follows
<span id="ALGO_VASP6">
</span></p>
<p class="mw-empty-elt">
</p>
<ul><li><a class="mw-selflink selflink">ALGO</a>=EVGW0 selects single-shot <i>G</i><sub>0</sub><i>W</i><sub>0</sub> calculations or partially self-consistent <i>GW</i> calculations. The orbitals (wavefunctions) of the previous groundstate calculations are maintained, and <a class="mw-redirect" href="/wiki/index.php/GW_calculations#G0W0" title="GW calculations">G0W0 calculations</a> are performed. If <a href="/wiki/index.php/NELM" title="NELM">NELM</a> is set, several iterations are performed, and the QP energies are updated in the calculation of <i>G</i> (for details, see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#gw0" title="GW calculations">EVGW0 calculations</a>).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=EVGW selects single-shot <i>G</i><sub>0</sub><i>W</i><sub>0</sub> calculations or partially self-consistent <i>GW</i> calculations. The orbitals of the previous groundstate calculations are maintained, and <a class="mw-redirect" href="/wiki/index.php/GW_calculations#G0W0" title="GW calculations">G0W0 calculations</a> are performed. If <a href="/wiki/index.php/NELM" title="NELM">NELM</a> is set, several iterations are performed, and the QP energies are updated in the calculation of <i>G</i> AND <i>W</i> (for details, see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#qpgw" title="GW calculations">self-consistent EVGW and QPGW calculations</a>).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=QPGW0 selects self-consistent <i>GW</i> calculations including off-diagonal components of the selfenergy. A full update of the QP energies AND one-electron orbitals is performed in the calculation of <i>G</i> only (for details see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#qpgw0" title="GW calculations">QPGW0 calculations</a>).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=QPGW selects self-consistent <i>GW</i> calculations, including off-diagonal components of the selfenergy. A full update of the QP energies AND one-electron orbitals is performed in the calculations of <i>G</i> AND <i>W</i> (for details, see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#qpgw" title="GW calculations">QPGW calculations</a>).</li></ul>
<p>Following tags are available as of VASP.6
</p>
<ul><li><a class="mw-selflink selflink">ALGO</a>=RPAR selects low scaling RPA total energy calculations (for details see <a href="/wiki/index.php/ACFDT/RPA_calculations#ACFDTR/RPAR" title="ACFDT/RPA calculations">ACFDT/RPA calculations</a>)</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=ACFDTR synonym for RPAR (for details see <a href="/wiki/index.php/ACFDT/RPA_calculations#ACFDTR/RPAR" title="ACFDT/RPA calculations">ACFDT/RPA calculations</a>)</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=ACFDTRK in combination with LMP2LT = .TRUE. selects the low scaling MP2 total energy calculations (for details see the <a href="/wiki/index.php/MP2_ground_state_calculation_-_Tutorial" title="MP2 ground state calculation - Tutorial">MP2 ground state Tutorial</a>)</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=GW0R selects self-consistent GW<sub>0</sub> calculations, where only the Green's function <i>G</i> is updated from the corresponding Dyson. The screened potential <i>W</i> remains unchanged after the first iteration. <a href="/wiki/index.php/NELM" title="NELM">NELM</a> iteration cycles are performed (see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#scGW0R" title="GW calculations">self-consistent GW calculations</a>).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=GWR selects self-consistent GW calculations, where both, <i>G</i> and <i>W</i> are updated from the corresponding Dyson equation. <a href="/wiki/index.php/NELM" title="NELM">NELM</a> iteration cycles are performed. (for details see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#scGWR" title="GW calculations">self-consistent GW calculations</a>).</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=G0W0R selects single-shot GW calculations, non-interacting <i>G</i> and <i>W</i> are determined from Kohn-Sham system and <a href="/wiki/index.php/NELM" title="NELM">NELM</a> tag is ignored. Use this tag for single-shot QP energies and first-order corrections to the density matrix (for details, see <a class="mw-redirect" href="/wiki/index.php/GW_calculations#G0W0R" title="GW calculations">single-shot GW calculations</a>).</li></ul>
<ul><li><a href="/wiki/index.php/NELMGW" title="NELMGW">NELMGW</a> replaces <a href="/wiki/index.php/NELM" title="NELM">NELM</a> in <a class="mw-redirect" href="/wiki/index.php/GW_calculations#scGWR" title="GW calculations">self-consistent GW calculations</a>.</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=CRPA selects <a href="/wiki/index.php/Constrained%E2%80%93random-phase%E2%80%93approximation_formalism" title="Constrained–random-phase–approximation formalism">constrained RPA alculations</a>.</li></ul>
<ul><li><a class="mw-selflink selflink">ALGO</a>=EVGW0R selects the low-scaling analog of EVGW0, that is the low-scaling partially self-consistent GW calculations, where non-interacting <i>G</i> and <i>W</i> are determined from Kohn-Sham system and <a href="/wiki/index.php/NELMGW" title="NELMGW">NELMGW</a> specifies the number of self-consistent loops for <i>G</i>. <i>W</i> is kept on the Kohn-Sham level.</li></ul>
