<p><a class="mw-selflink selflink">FMP_ACTIVE</a> = logical (aray) 
</p>
<p>Description: Select which atom types in the <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a>-file participate in swapping within the <a href="/wiki/index.php/M%C3%BCller-Plathe_method" title="Müller-Plathe method">Müller-Plathe method</a>.
</p>
<p><a class="mw-selflink selflink">FMP_ACTIVE</a> specifies whether or not (.TRUE. or .FALSE., respectively) an atomic type allowed for swapping within the <a href="/wiki/index.php/M%C3%BCller-Plathe_method" title="Müller-Plathe method">Müller-Plathe method</a>. One item for each of the atomic types defined in <a href="/wiki/index.php/POSCAR" title="POSCAR">POSCAR</a> must be supplied.
</p>
