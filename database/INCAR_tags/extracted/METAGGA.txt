<h2><span class="mw-headline" id="Aspherical_contributions_related_to_one-center_terms">Aspherical contributions related to one-center terms</span></h2>
<p><a href="/wiki/index.php/LASPH" title="LASPH">LASPH</a> =.TRUE. should be selected if a meta-GGA functional is selected. If <a href="/wiki/index.php/LASPH" title="LASPH">LASPH</a> =.FALSE.,
the one-center contributions are only calculated for a spherically averaged density and kinetic-energy
density. This means that the one-center contributions to the Kohn-Sham potential are also spherical. 
Since the PAW method describes the entire space using plane waves, errors are often small even
if the non-spherical contributions to the Kohn-Sham potential are neglected inside the PAW spheres
(additive augmentation, as opposed to the APW or FLAPW method where the plane wave contribution only
describes the interstitial region between the atoms). Anyhow, if the density is strongly non-spherical
around some atoms in your structure, <a href="/wiki/index.php/LASPH" title="LASPH">LASPH</a> =.TRUE. must be selected. Non-spherical terms are particularly encountered
in d- and f-elements, dimers, molecules, and solids with strong directional bonds.
</p>
<h2><span class="mw-headline" id="Convergence_issues">Convergence issues</span></h2>
<p>If convergence problems are encountered, it is recommended to preconverge the
calculations using the PBE functional and start the calculation from the <a href="/wiki/index.php/WAVECAR" title="WAVECAR">WAVECAR</a> file corresponding to the PBE ground state. Furthermore,
<a href="/wiki/index.php/ALGO" title="ALGO">ALGO</a> = A (conjugate gradient algorithm for orbitals) is often more stable
than charge density mixing, in particular if the system contains vacuum regions.
</p>
