<p><a class="mw-selflink selflink">TSUBSYS</a> = [real array] 
</p>
<p>Description: <a class="mw-selflink selflink">TSUBSYS</a> sets the temperatures for the atomic subsystems in calculations with multiple Anderson thermostats (in case VASP was compiled with <a class="mw-redirect" href="/wiki/index.php/Precompiler_flags" title="Precompiler flags">-Dtbdyn</a>).
</p>
<p>Up to three user-defined atomic subsystems may be coupled with independent Andersen thermostats<sup class="reference" id="cite_ref-Andersen80_1-0"><a href="#cite_note-Andersen80-1">[1]</a></sup> (<a href="/wiki/index.php/MDALGO" title="MDALGO">MDALGO</a>=13).
</p>
<p>The simulation temperature for the atomic subsystems is set by means of the <a class="mw-selflink selflink">TSUBSYS</a> tag (one has to specify one number for each subsystem).
</p>
