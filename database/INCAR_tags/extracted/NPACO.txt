<p><a class="mw-selflink selflink">NPACO</a> = [integer]<br/> Default: <b>NPACO</b> = 256 
</p>
<p>Description: <a class="mw-selflink selflink">NPACO</a> sets the number of slots in the pair-correlation function written to <a href="/wiki/index.php/PCDAT" title="PCDAT">PCDAT</a>.
</p>
<p>VASP evaluates the pair-correlation function each <a href="/wiki/index.php/NBLOCK" title="NBLOCK">NBLOCK</a> steps and writes the PC-function after <a href="/wiki/index.php/NBLOCK" title="NBLOCK">NBLOCK</a>×<a href="/wiki/index.php/KBLOCK" title="KBLOCK">KBLOCK</a> steps to the <a href="/wiki/index.php/PCDAT" title="PCDAT">PCDAT</a> file.
</p>
