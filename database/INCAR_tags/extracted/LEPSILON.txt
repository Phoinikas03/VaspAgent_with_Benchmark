<p><a class="mw-selflink selflink">LEPSILON</a> = .TRUE. | .FALSE.<br/> Default: <b>LEPSILON</b> = .FALSE. 
</p>
<p>Description: <a class="mw-selflink selflink">LEPSILON</a>=.TRUE. determines the static dielectric matrix, ion-clamped piezoelectric tensor and the Born effective charges using density functional perturbation theory.
</p>
<p>Determines the static ion-clamped dielectric matrix using density functional perturbation
theory. The dielectric matrix is calculated with and without local field effects. Usually local field effects are determined on the Hartree level, i.e. including changes of the Hartree potential.
To include microscopic changes of the exchange correlation potential 
the tag <a href="/wiki/index.php/LRPA" title="LRPA">LRPA</a>=.FALSE. must be set. The method is explained in detail in paper by Gajdoš <i>et al.</i>,<sup class="reference" id="cite_ref-gajdos:prb:06_1-0"><a href="#cite_note-gajdos:prb:06-1">[1]</a></sup> and closely follows the original work of Baroni and Resta.<sup class="reference" id="cite_ref-baroni:prb:86_2-0"><a href="#cite_note-baroni:prb:86-2">[2]</a></sup>
A summation over empty conduction band states is not required, instead the usual expressions in perturbation theory (<a href="/wiki/index.php/LOPTICS" title="LOPTICS">LOPTICS</a>=.TRUE.),
</p>

<p>It is important to note that exact values for the dielectric matrix are obtained even if only valence band states are calculated. Hence this method does not require to increase the <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a> parameter. <p>The values found after <tt>MACROSCOPIC STATIC DIELECTRIC TENSOR (excluding local field effects)</tt> should match exactly to the zero frequency values ω→0 determined by the method selected using <a href="/wiki/index.php/LOPTICS" title="LOPTICS">LOPTICS</a>=.TRUE..
This offers a convenient way to determine how many empty bands are required for
<a href="/wiki/index.php/LOPTICS" title="LOPTICS">LOPTICS</a>=.TRUE.. Simply execute VASP using <a class="mw-selflink selflink">LEPSILON</a>=.TRUE. in order to determine the exact values for the dielectric constants. Next, switch to <a href="/wiki/index.php/LOPTICS" title="LOPTICS">LOPTICS</a>=.TRUE. and increase the number of conduction bands until the same values are obtained as using density functional perturbation theory.
</p>
<p>Note that the routine also parses and uses the value supplied in the <a href="/wiki/index.php/LNABLA" title="LNABLA">LNABLA</a> tag.
Furthermore, the routine calculates the Born effective charge tensor (dynamical charges) and electronic contribution to the piezoelectric tensor ,
and prints them after

Regarding the piezoelectric tensor, mind the <a href="/wiki/index.php/ISIF" title="ISIF">definition/sign convention of the stress tensor</a>.

<p><br/>
We do not recommend to select <a href="/wiki/index.php/LOPTICS" title="LOPTICS">LOPTICS</a>=.TRUE. and <a class="mw-selflink selflink">LEPSILON</a>=.TRUE.
in a single run (although it might work in some versions).
Density functional perturbation theory <a class="mw-selflink selflink">LEPSILON</a>=.TRUE. does not require to increase <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a> and is, in fact, much slower if  <a href="/wiki/index.php/NBANDS" title="NBANDS">NBANDS</a> is increased, whereas the summation over emtpy conduction band states requires a large number of such states.
