</p>
<p>Description: This tag sets the filtering parameter for the angular filtering for <a href="/wiki/index.php/ML_IAFILT2" title="ML IAFILT2">ML_IAFILT2</a> in the machine learning force field method.
</p>
<p>This tag is only used if <a href="/wiki/index.php/ML_LAFILT2" title="ML LAFILT2">ML_LAFILT2</a>=<i>.TRUE.</i> and <a href="/wiki/index.php/ML_IAFILT2" title="ML IAFILT2">ML_IAFILT2</a>=2 are used.
</p>
