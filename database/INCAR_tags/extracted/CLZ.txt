<p><a class="mw-selflink selflink">CLZ</a> = [real] 
</p>
<p>Description: <a class="mw-selflink selflink">CLZ</a> selects a fractional or the full electron charge for the core hole in the core-excitation calculations with <a href="/wiki/index.php/ICORELEVEL" title="ICORELEVEL">ICORELEVEL</a>=2.
</p>
<p>In <a href="/wiki/index.php/Supercell_core-hole_theory" title="Supercell core-hole theory">SCH</a> <a class="mw-selflink selflink">CLZ</a> can be used to introduce a fraction of the core hole. For example, a half-core hole can be created to simulate a transition state in the X-ray excitation process.
</p>
