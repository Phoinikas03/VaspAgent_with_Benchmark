</p>
<p>Description: <a class="mw-selflink selflink">FERDO</a> sets the occupancies of the states in the down-spin channel for <a href="/wiki/index.php/ISMEAR" title="ISMEAR">ISMEAR</a>=-2 and <a href="/wiki/index.php/ISPIN" title="ISPIN">ISPIN</a>=2.
</p>
<p>To set the occupancies, specify
</p>
<p>The occupancies must be specified for all bands and k points. The band-index runs fastest. The occupancies must be between 0 and 1.
<a class="mw-selflink selflink">FERDO</a> has the same format as <a href="/wiki/index.php/FERWE" title="FERWE">FERWE</a>, please consider the notes on that page when setting <a class="mw-selflink selflink">FERDO</a>.
</p>
