<p><a class="mw-selflink selflink">IVDW_NL</a> = [integer] 
</p>
<p>Description: <a class="mw-selflink selflink">IVDW_NL</a> allows to select the kernel of the nonlocal van der Waals part of a functional (available as of VASP.6.4.0).
</p>
<p><a class="mw-selflink selflink">IVDW_NL</a>=1 corresponds to the kernel of Dion <i>et al.</i><sup class="reference" id="cite_ref-dion:prl:2004_1-0"><a href="#cite_note-dion:prl:2004-1">[1]</a></sup> and <a class="mw-selflink selflink">IVDW_NL</a>=2 to the kernel rVV10<sup class="reference" id="cite_ref-sabatini:prb:2013_2-0"><a href="#cite_note-sabatini:prb:2013-2">[2]</a></sup>. Note that the kernel of Dion <i>et al.</i> contains one adjustable parameter (<a href="/wiki/index.php/ZAB_VDW" title="ZAB VDW">ZAB_VDW</a>), while the rVV10 kernel contains two such parameters (<a href="/wiki/index.php/BPARAM" title="BPARAM">BPARAM</a> and <a href="/wiki/index.php/CPARAM" title="CPARAM">CPARAM</a>).
</p>
